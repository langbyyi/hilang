<h2 class="wp-block-heading">命令行工具</h2>



<h3 class="wp-block-heading">mysql（命令行客户端）</h3>



<p>功能：执行 SQL 语句、管理数据库，是最基础的交互工具。</p>



<p>连接示例：</p>





```
<code>mysql -u root -p -P 3306 -h localhost
</code>
```





<p><code>-u</code>指定用户名，<code>-p</code>用于输入密码，<code>-P</code>指定端口，<code>-h</code>指定主机。</p>



<h3 class="wp-block-heading">mysqldump（备份工具）</h3>



<p>功能：备份数据库或表结构及数据，支持多种参数定制备份内容。</p>



<p>示例：备份<code>test_db</code>库的<code>users</code>表：</p>





```
<code>mysqldump -u root -p test_db users &gt; D:/backups/users_backup.sql
</code>
```





<h3 class="wp-block-heading">mysqladmin（服务器管理工具）</h3>



<p>功能：管理 MySQL 服务器状态，如刷新权限、查看进程、关闭服务等。</p>



<p>示例：</p>





```
<code># 刷新权限
mysqladmin -u root -p flush-privileges

# 查看服务器状态
mysqladmin -u root -p status
</code>
```





<h3 class="wp-block-heading">mysqlpump（高效备份工具，MySQL 5.7+）</h3>



<p>功能：支持并行导出，备份大数据库时速度比 mysqldump 更快。</p>



<p>示例：备份<code>test_db</code>库：</p>





```
<code>mysqlpump -u root -p --databases test_db --output-file=test_db_backup.sql
</code>
```





<h2 class="wp-block-heading">图形化工具（新手推荐）</h2>



<ul class="wp-block-list">
<li><strong>Navicat</strong>：跨平台（Windows/macOS），支持可视化操作、SQL 编辑、数据导入导出，功能全面，适合开发和运维。</li>



<li><strong>phpMyAdmin</strong>：基于 Web 的管理工具，通过浏览器访问，适合中小型数据库管理，无需安装客户端。</li>



<li><strong>MySQL Workbench</strong>：官方工具，支持数据库设计、SQL 开发、性能监控，适合专业开发者。</li>
</ul>
