<p class="md-end-block md-p md-focus"><span class="md-pair-s md-expand"><strong><span class="md-plain">Nmap</span></strong></span><span class="md-plain">（Network Mapper，网络映射器）是一款功能强大的免费开源工具，常用于网络发现和安全审计。无论是扫描大型网络还是对单个主机进行详尽侦察，Nmap 都因其</span><span class="md-pair-s "><strong><span class="md-plain">灵活性</span></strong></span><span class="md-plain">和</span><span class="md-pair-s "><strong><span class="md-plain">详尽性</span></strong></span><span class="md-plain">而成为渗透测试和网络管理员的必备工具。</span></p>

<h3 class="md-end-block md-heading"><span class="md-plain">核心功能速览</span></h3>
<p class="md-end-block md-p"><span class="md-plain">Nmap 通过发送特殊构造的数据包并分析响应来工作，其核心功能包括：</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">主机发现</span></strong></span><span class="md-plain">：确定网络上哪些主机是活跃的。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">端口扫描</span></strong></span><span class="md-plain">：识别目标主机上开放的端口。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">服务和版本识别</span></strong></span><span class="md-plain">：识别端口上运行的具体服务（如 HTTP、SSH）及其版本号。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">操作系统识别</span></strong></span><span class="md-plain">：通过分析数据包响应的特征，推断目标主机的操作系统类型。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">脚本引擎（NSE）</span></strong></span><span class="md-plain">：Nmap 自带的脚本引擎，用于自动化执行多种任务，例如漏洞检测和配置错误检查。</span></p>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">基础使用方法</span></h3>
<p class="md-end-block md-p"><span class="md-plain">Nmap 是一款命令行工具，使用起来非常直观。</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">扫描单个 IP 地址</span></strong></span></p>



```
nmap 192.168.1.1
```


<p class="md-end-block md-p"><span class="md-plain">这会执行一个默认的 </span><span class="md-pair-s "><strong><span class="md-plain">TCP SYN 扫描</span></strong></span><span class="md-plain">，快速探测目标主机上的上千个常用端口。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">扫描一个 IP 段</span></strong></span></p>



```
nmap 192.168.1.1/24
```


<p class="md-end-block md-p"><span class="md-plain">这会扫描整个 C 类子网中的所有活跃主机。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">扫描指定端口</span></strong></span></p>



```
nmap -p 80,443,8080 192.168.1.1
```


<p class="md-end-block md-p"><span class="md-plain">使用 </span><span class="md-pair-s" spellcheck="false"><code>-p</code></span><span class="md-plain"> 参数指定要扫描的端口，可以加快扫描速度。</span></p>
</li>
</ul>
