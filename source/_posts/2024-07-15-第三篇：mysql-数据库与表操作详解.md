<h2 class="wp-block-heading">数据库基本操作（DDL）</h2>



<h3 class="wp-block-heading">查看所有数据库</h3>





```
<code>SHOW DATABASES;
</code>
```





<h3 class="wp-block-heading">创建数据库</h3>





```
<code>-- 直接创建（若数据库已存在会报错）
CREATE DATABASE 数据库名;

-- 推荐：判断不存在时再创建
CREATE DATABASE IF NOT EXISTS 数据库名;
</code>
```





<p>示例：创建名为<code>test_db</code>的数据库</p>





```
<code>CREATE DATABASE IF NOT EXISTS test_db;
</code>
```





<h3 class="wp-block-heading">删除数据库</h3>





```
<code>-- 直接删除（若数据库不存在会报错）
DROP DATABASE 数据库名;

-- 推荐：判断存在时再删除
DROP DATABASE IF EXISTS 数据库名;
</code>
```





<h3 class="wp-block-heading">数据存储位置</h3>



<p>MySQL 数据库默认存储在安装目录的<code>data</code>文件夹下，路径为<code>{MySQL安装目录}/data/{数据库名}/</code>，每个数据库对应一个独立文件夹。</p>

<h2 class="wp-block-heading">表的约束（保证数据合法性）</h2>



<p>约束是数据库的 “质检员”，用于防止无效数据进入表中，常见类型：</p>



<ul class="wp-block-list">
<li><strong>主键约束（PRIMARY KEY）</strong>：唯一标识一条记录，非空且唯一（一张表只能有一个主键）。</li>
</ul>





```
<code>id INT PRIMARY KEY  -- 例如用户ID
</code>
```





<ul class="wp-block-list">
<li><strong>非空约束（NOT NULL）</strong>：字段必须填写，不能为<code>NULL</code>。</li>
</ul>





```
<code>email VARCHAR(100) NOT NULL  -- 邮箱必须填写
</code>
```





<ul class="wp-block-list">
<li><strong>唯一约束（UNIQUE）</strong>：字段值不能重复（允许<code>NULL</code>，但<code>NULL</code>只允许出现一次）。</li>
</ul>





```
<code>username VARCHAR(50) UNIQUE  -- 用户名不能重复
</code>
```





<ul class="wp-block-list">
<li><strong>外键约束（FOREIGN KEY）</strong>：关联两张表，确保表间数据关系一致（如订单表的<code>user_id</code>关联用户表的<code>id</code>）。</li>
</ul>





```
<code>user_id INT, 
FOREIGN KEY(user_id) REFERENCES users(id)  -- 关联users表的id字段
</code>
```





<ul class="wp-block-list">
<li><strong>默认值约束（DEFAULT）</strong>：字段未填写时自动使用默认值。</li>
</ul>





```
<code>status VARCHAR(10) DEFAULT &#039;active&#039;  -- 状态默认值为“active”
</code>
```





<ul class="wp-block-list">
<li><strong>检查约束（CHECK）</strong>：限制字段值范围。</li>
</ul>





```
<code>age INT CHECK(age &gt;= 18)  -- 年龄必须≥18
</code>
```





<h2 class="wp-block-heading">表的创建与修改（DDL）</h2>



<h3 class="wp-block-heading">创建表</h3>





```
<code>CREATE TABLE 表名 (
  列名1 数据类型 约束条件,
  列名2 数据类型 约束条件,
  ...
) ENGINE=InnoDB DEFAULT CHARSET=utf8;  -- 引擎用InnoDB，字符集utf8
</code>
```





<p>示例：创建<code>users</code>表存储用户信息</p>





```
<code>CREATE TABLE users (
  id INT PRIMARY KEY AUTO_INCREMENT,  -- 自增主键
  username VARCHAR(50) NOT NULL UNIQUE,  -- 用户名非空且唯一
  email VARCHAR(100) NOT NULL,  -- 邮箱非空
  age INT CHECK(age &gt;= 18),  -- 年龄≥18
  register_time DATETIME DEFAULT CURRENT_TIMESTAMP  -- 注册时间默认当前时间
) ENGINE=InnoDB DEFAULT CHARSET=utf8;
</code>
```





<h3 class="wp-block-heading">删除表</h3>





```
<code>DROP TABLE 表名;  -- 删除表结构和所有数据（谨慎操作！）
</code>
```





<h3 class="wp-block-heading">修改表结构</h3>



<ul class="wp-block-list">
<li>添加字段：</li>
</ul>





```
<code>ALTER TABLE 表名 ADD COLUMN 列名 数据类型 约束;
-- 示例：给users表添加phone字段
ALTER TABLE users ADD COLUMN phone VARCHAR(20) UNIQUE;
</code>
```





<ul class="wp-block-list">
<li>修改字段类型：</li>
</ul>





```
<code>ALTER TABLE 表名 MODIFY COLUMN 列名 新数据类型;
-- 示例：将age字段改为SMALLINT
ALTER TABLE users MODIFY COLUMN age SMALLINT;
</code>
```





<ul class="wp-block-list">
<li>删除字段：</li>
</ul>





```
<code>ALTER TABLE 表名 DROP COLUMN 列名;
-- 示例：删除phone字段
ALTER TABLE users DROP COLUMN phone;
</code>
```





<ul class="wp-block-list">
<li>修改字段名：</li>
</ul>





```
<code>ALTER TABLE 表名 CHANGE COLUMN 旧列名 新列名 新数据类型;
-- 示例：将username改为user_name
ALTER TABLE users CHANGE COLUMN username user_name VARCHAR(50);
</code>
```





<ul class="wp-block-list">
<li>修改表名：</li>
</ul>





```
<code>RENAME TABLE 旧表名 TO 新表名;
-- 示例：将users改为user_info
RENAME TABLE users TO user_info;</code>
```
