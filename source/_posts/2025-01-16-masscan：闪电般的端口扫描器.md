<p class="md-end-block md-p md-focus"><span class="md-meta-i-c md-link md-expand"><a href="https://github.com/robertdavidgraham/masscan"><span class="md-plain">Masscan</span></a></span><span class="md-plain"> 是一款开源的</span><span class="md-pair-s "><strong><span class="md-plain">互联网级端口扫描器</span></strong></span><span class="md-plain">，以其</span><span class="md-pair-s "><strong><span class="md-plain">极高的扫描速度</span></strong></span><span class="md-plain">而闻名。它能在短短几分钟内扫描完整个 IPv4 地址空间的特定端口，被誉为“互联网级的 Nmap”。Masscan 的核心优势在于其异步传输技术，能够每秒发出数百万个数据包，非常适合大规模的网络资产普查。</span></p>

<h3 class="md-end-block md-heading"><span class="md-plain">核心特点与适用场景</span></h3>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">速度至上</span></strong></span><span class="md-plain">：Masscan 的首要目标就是快。它采用自定义的 TCP/IP 协议栈，不依赖操作系统内核，因此能够实现惊人的扫描速度。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">专注于端口扫描</span></strong></span><span class="md-plain">：与 Nmap 不同，Masscan 主要用于</span><span class="md-pair-s "><strong><span class="md-plain">快速发现开放端口</span></strong></span><span class="md-plain">，它不会进行复杂的服务版本识别或操作系统探测，这正是其速度的保证。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">高并发异步</span></strong></span><span class="md-plain">：Masscan 能够同时处理大量的网络请求，将扫描效率提升到极致。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">适用场景</span></strong></span><span class="md-plain">：</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">大规模资产普查</span></strong></span><span class="md-plain">：快速摸清全球或某个大型网络范围内，哪些主机开放了特定的端口，例如扫描所有暴露在公网上的 80 端口。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">安全研究</span></strong></span><span class="md-plain">：用于分析互联网上的服务分布、追踪新出现的网络威胁等。</span></p>
</li>
</ul>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">基础使用方法</span></h3>
<p class="md-end-block md-p"><span class="md-plain">Masscan 是一款命令行工具，使用起来非常直观，其语法与 Nmap 有些相似。</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">扫描单个 IP 地址的特定端口</span></strong></span></p>



```
masscan 192.168.1.1 -p80
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">扫描 IP 段的特定端口</span></strong></span></p>



```
masscan 192.168.1.0/24 -p80,443,8080
```


<p class="md-end-block md-p"><span class="md-plain">此命令会扫描 </span><span class="md-pair-s" spellcheck="false"><code>192.168.1.0</code></span><span class="md-plain"> 到 </span><span class="md-pair-s" spellcheck="false"><code>192.168.1.255</code></span><span class="md-plain"> 这个网段中所有主机的 80、443 和 8080 端口。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">扫描整个 IPv4 地址空间的特定端口</span></strong></span></p>



```
masscan 0.0.0.0/0 -p22 --rate 100000
```


<p class="md-end-block md-p"><span class="md-plain">这条命令会扫描全球所有主机的 22 端口，</span><span class="md-pair-s" spellcheck="false"><code>--rate</code></span><span class="md-plain"> 参数用于控制每秒发送的数据包数量，防止对网络造成过大压力。</span></p>
</li>
</ul>

<h3 class="md-end-block md-heading"><span class="md-plain">常用命令组合</span></h3>
<p class="md-end-block md-p"><span class="md-plain">这个命令组合旨在兼顾</span><span class="md-pair-s "><strong><span class="md-plain">扫描速度</span></strong></span><span class="md-plain">和</span><span class="md-pair-s "><strong><span class="md-plain">结果可用性</span></strong></span><span class="md-plain">，非常适合在资产普查时使用。</span></p>



```
masscan 192.168.1.0/24 -p80,443,8080,22,3389 --rate 100000 -oL web_ssh_scan.txt
```


<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">命令解析</span></strong></span><span class="md-plain">：</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-pair-s" spellcheck="false"><code>masscan 192.168.1.0/24</code></span></strong></span><span class="md-plain">：指定要扫描的目标 IP 段。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-pair-s" spellcheck="false"><code>-p80,443,8080,22,3389</code></span></strong></span><span class="md-plain">：指定要扫描的端口，包含常见的 Web 服务端口（80、443、8080）以及 SSH 和 RDP 端口（22、3389）。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-pair-s" spellcheck="false"><code>--rate 100000</code></span></strong></span><span class="md-plain">：控制每秒发送的数据包数量。</span></p>
</li>
 	<li class="md-list-item md-focus-container">
<p class="md-end-block md-p md-focus"><span class="md-pair-s "><strong><span class="md-pair-s" spellcheck="false"><code>-oL web_ssh_scan.txt</code></span></strong></span><span class="md-plain md-expand">：将扫描结果以列表格式保存到文件中。</span></p>
</li>
</ul>
</li>
</ul>
