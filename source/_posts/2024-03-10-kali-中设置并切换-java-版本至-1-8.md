<h2 class="md-end-block md-heading"><span class="md-plain">一、安装 JDK 1.8</span></h2>
<ol class="ol-list">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">下载 JDK 1.8 安装包</span></strong></span> <span class="md-plain">下载地址：</span><span class="md-pair-s" spellcheck="false"><code>https://repo.huaweicloud.com/java/jdk/8u202-b08/jdk-8u202-linux-x64.tar.gz</code></span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">创建目录并解压安装包</span></strong></span></p>



```bash
# 建立存放 Java 的目录
sudo mkdir -p /usr/local/java
# 复制安装包到目标目录
cp jdk-8u202-linux-x64.tar.gz /usr/local/java
# 进入目录并解压
cd /usr/local/java
sudo tar xzvf jdk-8u202-linux-x64.tar.gz
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">配置环境变量</span></strong></span> <span class="md-plain">编辑 </span><span class="md-pair-s" spellcheck="false"><code>/etc/profile</code></span><span class="md-plain"> 文件：</span></p>



```bash
sudo vim /etc/profile
```


<p class="md-end-block md-p"><span class="md-plain">在文件末尾添加以下内容（注意版本号与实际解压的目录一致）：</span></p>



```bash
JAVA_HOME=/usr/local/java/jdk1.8.0_202
PATH=$PATH:$HOME/bin:$JAVA_HOME/bin
export JAVA_HOME
export PATH
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">注册 Java 可执行文件</span></strong></span> <span class="md-plain">执行以下命令将 Java 相关命令添加到系统 alternatives 中：</span></p>



```bash
sudo update-alternatives --install "/usr/bin/java" "java" "/usr/local/java/jdk1.8.0_202/bin/java" 1
sudo update-alternatives --install "/usr/bin/javac" "javac" "/usr/local/java/jdk1.8.0_202/bin/javac" 1
sudo update-alternatives --install "/usr/bin/javaws" "javaws" "/usr/local/java/jdk1.8.0_202/bin/javaws" 1
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">重新加载环境变量</span></strong></span></p>



```bash
source /etc/profile
```


</li>
</ol>
<h2 class="md-end-block md-heading"><span class="md-plain">二、切换 Java 版本</span></h2>
<ol class="ol-list">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">查看并选择可用版本</span></strong></span> <span class="md-plain">执行以下命令进入版本选择界面：</span></p>



```bash
update-alternatives --config java
```


<p class="md-end-block md-p"><span class="md-plain">会显示类似以下的选项（示例）：</span></p>



```plaintext
有2个候选项可用于替换java(提供/usr/bin/java)。
​
选择       路径                                      优先级  状态
------------------------------------------------------------
0            /usr/lib/jvm/java-17-openjdk-amd64/bin/java   1111      自动模式
1            /usr/lib/jvm/java-17-openjdk-amd64/bin/java   1111      手动模式
2            /usr/local/java/jdk1.8.0_202/bin/java         1         手动模式
​
要维持当前值[*]请按<回车键>，或者键入选择的编号:
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">切换至 1.8 版本</span></strong></span> <span class="md-plain">输入 JDK 1.8 对应的编号（如示例中的 </span><span class="md-pair-s" spellcheck="false"><code>2</code></span><span class="md-plain">），按回车确认。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">验证版本</span></strong></span> <span class="md-plain">执行以下命令检查是否切换成功：</span></p>



```bash
java -version
```


<p class="md-end-block md-p md-focus"><span class="md-plain">若输出包含 </span><span class="md-pair-s" spellcheck="false"><code>java version &quot;1.8.0_202&quot;</code></span><span class="md-plain md-expand">，则表示切换成功。</span></p>
</li>
</ol>
