<h2 class="md-end-block md-heading"><span class="md-plain">一、工具介绍</span></h2>
<h3 class="md-end-block md-heading"><span class="md-plain">核心定位</span></h3>
<p class="md-end-block md-p"><span class="md-plain"><a href="https://github.com/SleepingBag945/dddd">dddd</a>是一款专为红队工作流优化的批量信息收集与供应链链漏洞漏洞探测工具，通过 "资产 -> 指纹 ->Poc-> 漏洞" 的流程设计，减少无效发包和机械操作，帮助安全人员高效完成网络资产梳理与安全评估。</span></p>

<h3 class="md-end-block md-heading"><span class="md-plain">核心特点</span></h3>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">自动识别输入类型</span></strong></span><span class="md-plain">：无需手动区分 IP、域名、URL 等目标类型</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">灵活的指纹识别</span></strong></span><span class="md-plain">：支持主动 / 被动指纹探测，支持与 / 或 / 非 / 括号等复杂逻辑运算</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">Nuclei v3 支持</span></strong></span><span class="md-plain">：兼容 Nuclei 格式的 PoC，内置 2400 + 漏洞检测规则</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">多源资产收集</span></strong></span><span class="md-plain">：集成 Hunter、Fofa、Quake 等网络空间搜索引擎，支持子域名枚举</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">低依赖特性</span></strong></span><span class="md-plain">：多系统开箱即用，v2.0 起可独立于 config 文件夹运行</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">丰富输出能力</span></strong></span><span class="md-plain">：支持 TXT、JSON、HTML 格式报告，内置审计日志功能</span></p>
</li>
</ul>
<h2 class="md-end-block md-heading"><span class="md-plain">二、基础使用指南</span></h2>
<h3 class="md-end-block md-heading md-focus"><span class="md-plain">最简命令</span></h3>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">扫描单个 IP：</span></p>



```bash
dddd -t 192.168.0.1
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">扫描网段：</span></p>



```bash
dddd -t 192.168.0.1/24
dddd -t 192.168.0.0-192.168.0.12
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">扫描网站：</span></p>



```bash
dddd -t http://test.com
```


</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">输出控制</span></h3>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">基本输出：默认保存为</span><span class="md-pair-s" spellcheck="false"><code>result.txt</code></span><span class="md-plain">，可通过</span><span class="md-pair-s" spellcheck="false"><code>-o</code></span><span class="md-plain">指定文件名</span></p>



```bash
dddd -t target.com -o output.txt
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">格式控制：通过</span><span class="md-pair-s" spellcheck="false"><code>-ot</code></span><span class="md-plain">指定输出格式（text/json）</span></p>



```bash
dddd -t target.com -ot json -o result.json
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">HTML 报告：默认生成时间戳命名的 HTML 文件，可通过</span><span class="md-pair-s" spellcheck="false"><code>-ho</code></span><span class="md-plain">指定</span></p>



```bash
dddd -t targets.txt -ho report.html
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">审计日志：</span><span class="md-pair-s" spellcheck="false"><code>-a</code></span><span class="md-plain">参数开启详细操作日志（保存至 audit.log）</span></p>



```bash
dddd -t 192.168.1.1 -a
```


</li>
</ul>
<h2 class="md-end-block md-heading"><span class="md-plain">三、核心功能使用</span></h2>
<h3 class="md-end-block md-heading"><span class="md-plain">1. 资产收集</span></h3>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">网络空间搜索引擎集成</span></strong></span><span class="md-plain">：</span></p>



```bash
# 使用Hunter搜索
dddd -t 'icp.name="xxxx有限公司"' -hunter -oip
​
# 同时使用Hunter和Fofa
dddd -t 'domain="example.com"' -hunter -fofa -o result.txt
```


<blockquote>
<p class="md-end-block md-p"><span class="md-plain">需在</span><span class="md-pair-s" spellcheck="false"><code>config</code></span><span class="md-plain">目录配置对应 API 密钥（如</span><span class="md-pair-s" spellcheck="false"><code>hunter.key</code></span><span class="md-plain">、</span><span class="md-pair-s" spellcheck="false"><code>fofa.key</code></span><span class="md-plain">）或通过环境变量设置</span></p>
</blockquote>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">子域名枚举</span></strong></span><span class="md-plain">：</span></p>



```bash
# 基础子域名枚举
dddd -t example.com -sd
​
# 自定义线程数（默认50）
dddd -t example.com -sd -sdt 100
```


<blockquote>
<p class="md-end-block md-p"><span class="md-plain">自定义字典可放入</span><span class="md-pair-s" spellcheck="false"><code>config/dict/subdomain.txt</code></span></p>
</blockquote>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">2. 指纹识别</span></h3>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">仅进行指纹识别（不扫描漏洞）：</span></p>



```bash
dddd -t http://www.xxx.com -npoc
dddd -t 172.16.100.11:3307 -npoc
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">精准指纹匹配（支持复杂逻辑）：</span></p>



```bash
# 识别"Apache Tomcat 9.x且开启管理端"的资产
dddd -t http://target.com -fp '("Apache Tomcat 9" && "manager/html")'
```


</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">3. 漏洞扫描</span></h3>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">基于历史结果扫描：</span></p>



```bash
# 导入之前的指纹识别结果直接进行漏洞扫描
dddd -t result.txt
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">自定义 PoC 使用：</span></p>



```bash
# 指定自定义PoC文件夹
dddd -t target.txt -nt ./my-pocs
```


<blockquote>
<p class="md-end-block md-p"><span class="md-plain">自定义 PoC 需为 Nuclei 格式，存放于</span><span class="md-pair-s" spellcheck="false"><code>config/pocs</code></span><span class="md-plain">目录时会自动加载（优先级高于内置）</span></p>
</blockquote>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">限速扫描（敏感环境适用）：</span></p>



```bash
# 限制每秒最多10个请求
dddd -t 172.16.0.0/16 -rate 10
```


</li>
</ul>
<h2 class="md-end-block md-heading"><span class="md-plain">四、场景化使用示例</span></h2>
<h3 class="md-end-block md-heading"><span class="md-plain">红队外网探测</span></h3>


```bash
# 基于备案信息收集资产并导出IP
dddd -t 'icp.name="某科技公司"' -hunter -fofa -oip -o outer_assets.txt
​
# 低感知模式（减少主动发包）
dddd -t 'icp.name="目标企业"' -hunter -low
```


<h3 class="md-end-block md-heading"><span class="md-plain">内网渗透扫描</span></h3>


```bash
# 扫描内网网段（仅指纹识别）
dddd -t 10.0.0.0/8 -port 1-65535 -npoc
​
# 完整扫描（包含漏洞检测）
dddd -t 192.168.1.0/24 -o intranet_results.txt
```


<h3 class="md-end-block md-heading"><span class="md-plain">特定漏洞验证</span></h3>


```bash
# 使用专用PoC扫描Log4j漏洞
dddd -t targets.txt -nt ./log4j-pocs -o log4j_results.txt
```
