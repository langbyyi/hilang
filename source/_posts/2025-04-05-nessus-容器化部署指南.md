<h2 class="md-end-block md-heading"><span class="md-plain">1. 环境准备与容器部署</span></h2>
<h3 class="md-end-block md-heading"><span class="md-plain">1.1 安装 Docker（如未安装）</span></h3>


```bash
apt-get update && apt-get install -y docker.io && systemctl start docker && systemctl enable docker
```


<h3 class="md-end-block md-heading"><span class="md-plain">1.2 拉取并启动 Nessus 容器</span></h3>


```bash
# 启动容器并映射端口8834
docker run -itd --name=ramisec_nessus -p 8834:8834 ramisec/nessus
​
# 验证容器状态
docker ps | grep ramisec_nessus  # 显示容器信息即为成功
```


<h2 class="md-end-block md-heading"><span class="md-plain">2. 获取关键信息</span></h2>
<h3 class="md-end-block md-heading"><span class="md-plain">2.1 申请激活码</span></h3>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p md-focus"><span class="md-plain">访问官网：</span><span class="md-meta-i-c md-link"><a href="https://www.tenable.com/products/nessus/nessus-essentials"><span class="md-plain">Nessus Essentials</span></a></span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">填写信息后，查收邮箱获取激活码（示例：</span><span class="md-pair-s" spellcheck="false"><code>NU8L-AKY5-P37Q-5CDS-7DP8</code></span><span class="md-plain">）</span></p>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">2.2 生成挑战码</span></h3>


```bash
docker exec -it ramisec_nessus /opt/nessus/sbin/nessuscli fetch --challenge
```

<h3 class="md-end-block md-heading"><span class="md-plain">2.3 获取插件更新链接</span></h3>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">打开离线更新页面：</span><span class="md-link md-pair-s" spellcheck="false"><a href="https://plugins.nessus.org/v2/offline.php">https://plugins.nessus.org/v2/offline.php</a></span></p>
<p class="md-end-block md-p"><span class="md-image md-img-loaded" data-src="./image-20250805175907356.png"><img class="aligncenter wp-image-673 size-large" src="/wp-content/uploads/2025/img-89d702fd9c50.png" alt="" width="1024" height="750" /></span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">输入激活码和挑战码，提交后复制生成的插件链接（格式：</span><span class="md-pair-s" spellcheck="false"><code>https://plugins.nessus.org/v2/...</code></span><span class="md-plain">）</span></p>
</li>
</ul>
<h2 class="md-end-block md-heading"><span class="md-plain">3. 一键更新插件（核心步骤）</span></h2>
<p class="md-end-block md-p"><span class="md-plain">将获取的插件链接替换到以下命令中执行（约等待20分钟）：</span></p>



```bash
docker exec -it ramisec_nessus /bin/bash -c "/nessus/update.sh 'https://plugins.nessus.org/v2/...'"
```

<h2 class="md-end-block md-heading"><span class="md-plain">4. 登录与使用</span></h2>
<h3 class="md-end-block md-heading"><span class="md-plain">4.1 获取 / 重置密码</span></h3>


```bash
# 查看初始密码
docker logs ramisec_nessus | grep -i "password"
​
# 若未找到，手动重置（按提示输入新密码）
docker exec -it ramisec_nessus /opt/nessus/sbin/nessuscli chpasswd admin
```
