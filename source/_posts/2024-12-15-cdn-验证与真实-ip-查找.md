<h2 class="md-end-block md-heading md-focus"><span class="md-plain md-expand">一、CDN 存在性验证</span></h2>
<p class="md-end-block md-p"><span class="md-plain">验证核心逻辑为：若域名解析对应 IP 不唯一，则大概率使用 CDN，主要通过以下两种方法实现。</span></p>

<h2 class="md-end-block md-heading"><span class="md-plain">二、绕过 CDN 查找真实 IP 的核心方法</span></h2>
<h3 class="md-end-block md-heading"><span class="md-plain">（一）查询历史 DNS 记录</span></h3>
<p class="md-end-block md-p"><span class="md-plain">通过获取域名与 IP 绑定的历史数据，或分析当前可能泄漏 IP 的 DNS 记录（如 MX 记录），定位真实 IP。</span></p>

<ol class="ol-list" start="">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">历史记录查询渠道</span></strong></span><span class="md-plain">：包括</span><span class="md-link md-pair-s" spellcheck="false"><a href="https://dnsdb.io/zh-cn/">https://dnsdb.io/zh-cn/</a></span><span class="md-plain">、</span><span class="md-link md-pair-s" spellcheck="false"><a href="https://x.threatbook.cn/">https://x.threatbook.cn/</a></span><span class="md-plain">、</span><span class="md-link md-pair-s" spellcheck="false"><a href="http://toolbar.netcraft.com/site_report?url=">http://toolbar.netcraft.com/site_report?url=</a></span><span class="md-plain">等平台，以及 </span><span class="md-link md-pair-s" spellcheck="false"><a href="https://securitytrails.com/">https://securitytrails.com</a></span><span class="md-plain">。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">当前记录泄漏点</span></strong></span><span class="md-plain">：若网站邮件服务器与 Web 服务器在同一 IP，MX 记录会直接暴露真实 IP 。</span></p>
</li>
</ol>
<h3 class="md-end-block md-heading"><span class="md-plain">（二）查询子域名</span></h3>
<p class="md-end-block md-p"><span class="md-plain">因 CDN 成本较高，站长多仅为主站 / 高流量子站部署 CDN，小流量子站可能与主站同服务器 / C 段，可通过子域名 IP 推导真实 IP。</span></p>

<ol class="ol-list" start="">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">子域名获取方式</span></strong></span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">微步在线：输入域名（如</span><span class="md-meta-i-c  md-link"><a href="https://baidu.com/"><span class="md-plain">baidu.com</span></a></span><span class="md-plain">）点击 “子域名” 选项，免费用户每月 5 次查询机会 。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">Dnsdb：输入</span><span class="md-pair-s" spellcheck="false"><code>域名 type:A</code></span><span class="md-plain">（如</span><span class="md-meta-i-c  md-link"><a href="https://baidu.com/"><span class="md-plain">baidu.com</span></a></span><span class="md-plain"> type:A）收集子域名及 IP 。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">Google 搜索：用</span><span class="md-pair-s" spellcheck="false"><code>site:域名 -www</code></span><span class="md-plain">（如site:baidu.com -www）筛选非 www 子域名 。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">扫描工具：子域名挖掘机（基于字典挖掘）、subdomainbrute（Windows 下 CMD 执行</span><span class="md-pair-s" spellcheck="false"><code>python subdomainbrute.py 域名 --full</code></span><span class="md-plain">） 。</span></p>
</li>
</ul>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">后续操作</span></strong></span><span class="md-plain">：解析子域名 IP，用非 CDNIP 尝试解析主站，获取真实 IP 。</span></p>
</li>
</ol>
<h3 class="md-end-block md-heading"><span class="md-plain">（三）网络空间引擎搜索法</span></h3>
<p class="md-end-block md-p"><span class="md-plain">通过网站标题（title）、页面内容（body）等独特特征，在引擎中匹配对应 IP。</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">常用引擎：Shodan、FOFA、钟馗之眼 。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">搜索语法：FOFA 输入</span><span class="md-pair-s" spellcheck="false"><code>title:&quot;网站title关键字&quot;</code></span><span class="md-plain">或</span><span class="md-pair-s" spellcheck="false"><code>body:&quot;网站body特征&quot;</code></span><span class="md-plain"> 。</span></p>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">（四）网站邮件订阅查找</span></h3>
<p class="md-end-block md-p"><span class="md-plain">网站 RSS 邮件订阅通过</span><span class="md-pair-s" spellcheck="false"><code>sendmail</code></span><span class="md-plain">发送邮件，邮件源码中会包含服务器真实 IP，订阅后查看源码即可 。</span></p>

<h3 class="md-end-block md-heading"><span class="md-plain">（五）利用 SSL 证书查找</span></h3>
<p class="md-end-block md-p"><span class="md-plain">Web 服务器需配置 SSL 证书，攻击者可通过扫描全网证书匹配域名，定位真实 IP。</span></p>

<ol class="ol-list" start="">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">核心工具</span></strong></span><span class="md-plain">：Censys（每日扫描 IPv4 地址空间，收集设备、证书信息） 。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">操作流程</span></strong></span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">已知域名：构造搜索参数</span><span class="md-pair-s" spellcheck="false"><code>parsed.names:域名 and tags.raw:trusted</code></span><span class="md-plain">（如parsed.names:域名 and tags.raw:trusted），点击证书 “Explore” 查看关联 IPv4 主机 。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">已知证书（如.onion 服务）：提取证书 “SHA1 fingerprint”，在 Censys IPv4 主机搜索框粘贴匹配 。</span></p>
</li>
</ul>
</li>
</ol>
<h3 class="md-end-block md-heading"><span class="md-plain">（六）使用国外主机解析域名</span></h3>
<p class="md-end-block md-p"><span class="md-plain">国内 CDN 多仅覆盖国内线路，国外主机直接访问域名可能获取真实 IP 。</span></p>

<h3 class="md-end-block md-heading"><span class="md-plain">（七）网站漏洞查找</span></h3>
<p class="md-end-block md-p"><span class="md-plain">利用漏洞获取敏感信息或权限，间接找到真实 IP。</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">常见漏洞：敏感文件泄露（如 phpinfo 探针）、XSS 盲打、SSRF 等；或获取 CDN 管理员账号查看配置 。</span></p>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">（八）F5 LTM 解码法</span></h3>
<p class="md-end-block md-p"><span class="md-plain">若服务器使用 F5 LTM 负载均衡，可通过</span><span class="md-pair-s" spellcheck="false"><code>Set-Cookie</code></span><span class="md-plain">字段解码获取真实 IP，以</span><span class="md-pair-s" spellcheck="false"><code>Set-Cookie:BIGipServerpool_8.29_8030=487098378.24095.0000</code></span><span class="md-plain">为例：</span></p>

<ol class="ol-list" start="">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">提取第一小节十进制数：487098378 。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">转换为十六进制：1d08880a 。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">从后至前每四位分组：0a.88.08.1d 。</span></p>
</li>
 	<li class="md-list-item md-focus-container">
<p class="md-end-block md-p md-focus"><span class="md-plain md-expand">每组转为十进制：10.136.8.29（即真实 IP） 。</span></p>
</li>
</ol>
