<h4 class="md-end-block md-heading"><span class="md-plain">说明</span></h4>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">ASCII 码（American Standard Code for Information Interchange）是基于拉丁字母的编码系统，用于表示文本信息；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">0-127 为标准 ASCII 码，涵盖控制字符、数字、字母及常见符号；</span></p>
</li>
 	<li class="md-list-item md-focus-container">
<p class="md-end-block md-p md-focus"><span class="md-plain md-expand">128-255 为扩展 ASCII 码，包含特殊符号、货币符、重音字母等，适用于多语言场景。</span></p>
</li>
</ul>
