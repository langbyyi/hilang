<p class="md-end-block md-p md-focus"><span class="md-meta-i-c md-link md-expand"><a href="https://github.com/shadow1ng/fscan"><span class="md-plain">fscan</span></a></span><span class="md-plain"> 是由 </span><span class="md-pair-s "><strong><span class="md-plain">shadow1ng</span></strong></span><span class="md-plain"> 开发的一款功能强大的</span><span class="md-pair-s "><strong><span class="md-plain">内网综合扫描工具</span></strong></span><span class="md-plain">。它集成了多种扫描功能，旨在帮助安全研究人员和渗透测试人员快速、自动化地完成内网信息收集和漏洞发现。</span></p>

<h4 class="md-end-block md-heading"><span class="md-plain">1. 核心功能速览</span></h4>
<p class="md-end-block md-p"><span class="md-pair-s" spellcheck="false"><code>fscan</code></span><span class="md-plain"> 的最大特点是</span><span class="md-pair-s "><strong><span class="md-plain">自动化</span></strong></span><span class="md-plain">和</span><span class="md-pair-s "><strong><span class="md-plain">多功能集成</span></strong></span><span class="md-plain">。它能一键完成以下任务：</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">端口扫描</span></strong></span><span class="md-plain">：快速探测目标主机的开放端口。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">服务识别</span></strong></span><span class="md-plain">：识别开放端口上运行的具体服务（如 HTTP、SSH）及其版本，为后续的漏洞利用提供依据。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">弱口令检测</span></strong></span><span class="md-plain">：自动对常见服务（如 SSH、MySQL、Redis）进行弱口令字典爆破。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">漏洞扫描</span></strong></span><span class="md-plain">：内置多种常见漏洞检测模块，例如</span><span class="md-pair-s "><strong><span class="md-plain">永恒之蓝</span></strong></span><span class="md-plain">（MS17-010）、</span><span class="md-pair-s "><strong><span class="md-plain">Weblogic</span></strong></span><span class="md-plain">、</span><span class="md-pair-s "><strong><span class="md-plain">Struts2</span></strong></span><span class="md-plain"> 等中间件漏洞，以及</span><span class="md-pair-s "><strong><span class="md-plain">Redis</span></strong></span><span class="md-plain">、</span><span class="md-pair-s "><strong><span class="md-plain">Elasticsearch</span></strong></span><span class="md-plain"> 等数据库的未授权访问漏洞。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">信息收集</span></strong></span><span class="md-plain">：扫描内网存活主机、获取主机名、操作系统等基本信息。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p md-focus"><span class="md-pair-s "><strong><span class="md-plain">敏感文件扫描</span></strong></span><span class="md-plain">：扫描 Web 路径下的敏感文件，如 </span><span class="md-pair-s" spellcheck="false"><code>.git</code></span><span class="md-plain"> 泄露、</span><span class="md-pair-s" spellcheck="false"><code>phpinfo.php</code></span><span class="md-plain"> 等。</span></p>
</li>
</ul>
<h4 class="md-end-block md-heading"><span class="md-plain">2. 基础使用方法</span></h4>
<p class="md-end-block md-p"><span class="md-pair-s" spellcheck="false"><code>fscan</code></span><span class="md-plain"> 是一款命令行工具，使用方法非常简单，通常只需要指定目标 IP 或 IP 段。</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">扫描单个 IP 地址</span></strong></span></p>



```
fscan -h 192.168.1.1
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">扫描一个 IP 段</span></strong></span></p>



```
fscan -h 192.168.1.1/24
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">扫描多个不连续 IP</span></strong></span></p>



```
fscan -h 192.168.1.1,192.168.1.10-20
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">指定特定端口进行扫描</span></strong></span></p>



```
fscan -h 192.168.1.1/24 -p 22,80,3306
```


</li>
</ul>
