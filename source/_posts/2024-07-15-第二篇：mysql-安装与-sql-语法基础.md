<h2 class="wp-block-heading">MySQL 安装指南（新手友好）</h2>



<h3 class="wp-block-heading">推荐安装方式</h3>



<p>对于新手，<strong>phpStudy 集成开发环境</strong>是最优选择（Windows 系统），它包含 Apache 服务器、MySQL 数据库和 PHP 语言环境，一键安装即可使用，无需复杂配置。</p>



<ul class="wp-block-list">
<li>安装路径<strong>必须避免中文和空格</strong>（例如<code>D:\phpstudy_pro</code>，而非<code>D:\我的软件\php study</code>），否则可能导致服务启动失败。</li>
</ul>



<h3 class="wp-block-heading">登录 MySQL 的详细步骤</h3>



<ol class="wp-block-list">
<li>找到 MySQL 安装目录的<code>bin</code>文件夹，例如<code>D:\phpstudy_pro\Extensions\MySQL5.7.26\bin</code>；</li>



<li>在文件夹地址栏输入<code>cmd</code>，按回车打开命令行窗口；</li>



<li>输入登录命令：</li>
</ol>





```
<code>mysql -h 主机地址 -u 用户名 -p</code>
```





<p><code>-h</code>指定主机（默认<code>localhost</code>，可省略），<code>-u</code>指定用户名（默认<code>root</code>），<code>-p</code>用于输入密码（回车后输入更安全），<code>-P</code>指定端口（默认 3306）。本地登录<code>root</code>用户可直接输入<code>mysql -u root -p</code>，输入密码后即可进入 MySQL 交互界面。</p>



<h2 class="wp-block-heading">SQL 语句的四大分类</h2>



<p>SQL（结构化查询语言）是操作数据库的通用语言，按功能分为四大类：</p>



<ul class="wp-block-list">
<li><strong>DDL（数据定义语言）</strong>：用于创建、修改或删除数据库和表结构，核心命令：<code>CREATE</code>（创建）、<code>ALTER</code>（修改）、<code>DROP</code>（删除）。</li>



<li><strong>DML（数据操作语言）</strong>：负责数据的增删改，核心命令：<code>INSERT</code>（插入）、<code>UPDATE</code>（更新）、<code>DELETE</code>（删除）。</li>



<li><strong>DQL（数据查询语言）</strong>：用于查询数据，核心命令：<code>SELECT</code>（查询），可结合条件、排序等实现复杂查询。</li>



<li><strong>DCL（数据控制语言）</strong>：用于权限管理和事务控制，核心命令：<code>GRANT</code>（授权）、<code>REVOKE</code>（回收权限）、<code>COMMIT</code>（提交事务）、<code>ROLLBACK</code>（回滚事务）。</li>
</ul>



<h2 class="wp-block-heading">SQL 语句书写规范（必知）</h2>



<ul class="wp-block-list">
<li>每条语句必须以 <code>;</code> 结尾（例如<code>SELECT * FROM users;</code>）；</li>



<li>语句不区分大小写（<code>SELECT</code>与<code>select</code>效果相同），但建议关键字大写（如<code>SELECT</code>、<code>INSERT</code>），增强可读性；</li>



<li>单词之间需用<strong>半角空格或换行</strong>分隔，例如：</li>
</ul>





```
<code>SELECT username, age FROM users WHERE age &gt; 18;</code>
```





<h3 class="wp-block-heading">常数书写规则</h3>



<ul class="wp-block-list">
<li><strong>整数</strong>：直接书写，如<code>123</code>、<code>-456</code>；</li>



<li><strong>小数</strong>：带小数点，如<code>3.14</code>、<code>-0.5</code>；</li>



<li><strong>科学计数</strong>：如<code>1.2E3</code>（表示 1200）、<code>5e-2</code>（表示 0.05）；</li>



<li><strong>字符串</strong>：用单引号或双引号包裹，如<code>&#039;张三&#039;</code>、<code>&quot;hello&quot;</code>；</li>



<li><strong>日期时间</strong>：严格遵循<code>&#039;YYYY-MM-DD&#039;</code>或<code>&#039;YYYY-MM-DD HH:MM:SS&#039;</code>格式，如<code>&#039;2025-07-18&#039;</code>、<code>&#039;2025-07-18 15:30:00&#039;</code>；</li>



<li><strong>布尔值</strong>：<code>TRUE</code>等价于 1，<code>FALSE</code>等价于 0。</li>
</ul>
