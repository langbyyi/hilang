<h2 class="md-end-block md-heading"><span class="md-plain">二、用户配置</span></h2>
<p class="md-end-block md-p"><span class="md-plain">提交代码前必须配置身份，否则无法正常提交。</span></p>

<figure class="md-table-fig table-figure">
<table class="md-table">
<thead>
<tr class="md-end-block">
<th><span class="td-span"><span class="md-plain">命令</span></span></th>
<th><span class="td-span"><span class="md-plain">说明</span></span></th>
</tr>
</thead>
<tbody>
<tr class="md-end-block">
<td><span class="td-span"><span class="md-plain">git config user.name <名字></span></span></td>
<td><span class="td-span"><span class="md-plain">当前仓库配置用户名</span></span></td>
</tr>
<tr class="md-end-block">
<td><span class="td-span"><span class="md-plain">git config --global user.name <名字></span></span></td>
<td><span class="td-span"><span class="md-plain">全局配置用户名（所有仓库生效）</span></span></td>
</tr>
<tr class="md-end-block">
<td><span class="td-span"><span class="md-plain">git config --global user.email <邮箱></span></span></td>
<td><span class="td-span"><span class="md-plain">全局配置用户邮箱</span></span></td>
</tr>
<tr class="md-end-block">
<td><span class="td-span"><span class="md-plain">git config --global alias.<别名> < 命令 ></span></span></td>
<td><span class="td-span"><span class="md-plain">自定义 Git 命令别名，简化输入</span></span></td>
</tr>
<tr class="md-end-block">
<td><span class="td-span"><span class="md-plain">git config --system core.editor <编辑器></span></span></td>
<td><span class="td-span"><span class="md-plain">设置系统默认文本编辑器</span></span></td>
</tr>
<tr class="md-end-block">
<td><span class="td-span"><span class="md-plain">git config --global --edit</span></span></td>
<td><span class="td-span"><span class="md-plain">打开并手动编辑全局配置文件</span></span></td>
</tr>
</tbody>
</table>
</figure>

<h2 class="md-end-block md-heading"><span class="md-plain">三、工作区、暂存区与提交</span></h2>
<p class="md-end-block md-p"><span class="md-plain">最常用的核心流程：修改 → 暂存 → 提交。</span></p>

<figure class="md-table-fig table-figure">
<table class="md-table">
<thead>
<tr class="md-end-block">
<th><span class="td-span"><span class="md-plain">命令</span></span></th>
<th><span class="td-span"><span class="md-plain">说明</span></span></th>
</tr>
</thead>
<tbody>
<tr class="md-end-block">
<td><span class="td-span"><span class="md-plain">git add <目录 / 文件></span></span></td>
<td><span class="td-span"><span class="md-plain">将文件或目录的修改加入暂存区</span></span></td>
</tr>
<tr class="md-end-block">
<td><span class="td-span"><span class="md-plain">git commit -m "提交信息"</span></span></td>
<td><span class="td-span"><span class="md-plain">提交暂存区内容，附带说明</span></span></td>
</tr>
<tr class="md-end-block">
<td><span class="td-span"><span class="md-plain">git status</span></span></td>
<td><span class="td-span"><span class="md-plain">查看文件状态：已暂存、未暂存、未跟踪</span></span></td>
</tr>
<tr class="md-end-block">
<td><span class="td-span"><span class="md-plain">git log</span></span></td>
<td><span class="td-span"><span class="md-plain">查看完整提交历史记录</span></span></td>
</tr>
<tr class="md-end-block">
<td><span class="td-span"><span class="md-plain">git commit --amend -m "新信息"</span></span></td>
<td><span class="td-span"><span class="md-plain">将当前暂存修改合并到上一次提交，并更新提交信息</span></span></td>
</tr>
</tbody>
</table>
</figure>

<h2 class="md-end-block md-heading"><span class="md-plain">五、撤销与重置</span></h2>
<p class="md-end-block md-p"><span class="md-plain">误操作、写坏代码时，用这些命令安全回滚。</span></p>

<figure class="md-table-fig table-figure">
<table class="md-table">
<thead>
<tr class="md-end-block">
<th><span class="td-span"><span class="md-plain">命令</span></span></th>
<th><span class="td-span"><span class="md-plain">说明</span></span></th>
</tr>
</thead>
<tbody>
<tr class="md-end-block">
<td><span class="td-span"><span class="md-plain">git revert <提交 ID></span></span></td>
<td><span class="td-span"><span class="md-plain">新建提交撤销指定提交，不破坏历史（安全）</span></span></td>
</tr>
<tr class="md-end-block">
<td><span class="td-span"><span class="md-plain">git reset <文件></span></span></td>
<td><span class="td-span"><span class="md-plain">将文件移出暂存区，不修改工作区</span></span></td>
</tr>
<tr class="md-end-block">
<td><span class="td-span"><span class="md-plain">git reset</span></span></td>
<td><span class="td-span"><span class="md-plain">清空暂存区，保留工作区修改</span></span></td>
</tr>
<tr class="md-end-block">
<td><span class="td-span"><span class="md-plain">git reset --hard</span></span></td>
<td><span class="td-span"><span class="md-plain">清空暂存区并丢弃所有工作区修改（慎用）</span></span></td>
</tr>
<tr class="md-end-block">
<td><span class="td-span"><span class="md-plain">git reset <提交 ID></span></span></td>
<td><span class="td-span"><span class="md-plain">回退到指定提交，保留工作区</span></span></td>
</tr>
<tr class="md-end-block">
<td><span class="td-span"><span class="md-plain">git reset <提交 ID> --hard</span></span></td>
<td><span class="td-span"><span class="md-plain">强制回退到指定提交，清空所有后续修改</span></span></td>
</tr>
</tbody>
</table>
</figure>

<figure class="md-table-fig table-figure">
<table class="md-table">
<thead>
<tr class="md-end-block">
<th><span class="td-span"><span class="md-plain">命令</span></span></th>

<tbody>

<tr class="md-end-block">
<td><span class="td-span"><span class="md-plain">git push <远程> --tags</span></span></td>
<td><span class="td-span"><span class="md-plain">推送所有本地标签到远程</span></span></td>
