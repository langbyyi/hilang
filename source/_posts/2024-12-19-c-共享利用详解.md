<div class="auto-hide-last-sibling-br paragraph-JOTKXA paragraph-element br-paragraph-space">在 Windows 系统中，C$<span class="container-X8VMJb math-inline" data-custom-copy-text="\(共享是一种默认开启的 administrative 共享（管理员共享），主要用于系统管理员远程管理计算机。合理利用C\)">共享是一种默认开启的 administrative 共享（管理员共享），主要用于系统管理员远程管理计算机。合理利用C$</span>共享可实现文件传输、程序执行等操作，但同时也可能成为安全风险点。本文将从基础挂载到高级利用，全面解析 C$ 共享的操作方法与安全注意事项。</div>
<h2 class="header-vfC6AV auto-hide-last-sibling-br">一、C$<span class="container-X8VMJb math-inline" data-custom-copy-text="\(共享基础：什么是C\)">共享基础：什么是C$</span>共享？</h2>
<div class="auto-hide-last-sibling-br paragraph-JOTKXA paragraph-element br-paragraph-space">C$<span class="container-X8VMJb math-inline" data-custom-copy-text="\(共享是Windows系统为管理员预留的默认隐藏共享，对应系统的C盘根目录，仅允许具有管理员权限的用户访问。类似的默认共享还包括D\)">共享是Windows系统为管理员预留的默认隐藏共享，对应系统的C盘根目录，仅允许具有管理员权限的用户访问。类似的默认共享还包括D$（</span>D 盘）、admin$<span class="container-X8VMJb math-inline" data-custom-copy-text="\(（系统目录）等，这些共享通过"\)">（系统目录）等，这些共享通过"$</span>" 符号隐藏，无法直接在资源管理器中显示，需通过特定命令或路径访问。</div>
<div class="auto-hide-last-sibling-br paragraph-JOTKXA paragraph-element br-paragraph-space"><strong>核心特点</strong>：</div>
<ul class="auto-hide-last-sibling-br">
 	<li>仅管理员权限可访问，普通用户无权限挂载或操作；</li>
 	<li>默认随系统启动，无需手动开启；</li>
 	<li>隐藏属性，需通过<code>\\目标IP\C$</code>路径访问。</li>
</ul>
<h2 class="header-vfC6AV auto-hide-last-sibling-br">二、挂载 C$ 共享：连接远程 C 盘的关键步骤</h2>
<div class="auto-hide-last-sibling-br paragraph-JOTKXA paragraph-element br-paragraph-space">挂载 C<span class="container-X8VMJb math-inline" data-custom-copy-text="\(共享是实现远程操作的前提，需通过`net use`命令建立本地盘符与远程C\)">$共享是实现远程操作的前提，需通过`net use`命令建立本地盘符与远程C$</span>共享的映射。</div>
<h3 class="header-vfC6AV auto-hide-last-sibling-br">1. 挂载命令格式</h3>
<div class="relative w-fit custom-code-block-canvas-wrapper w-full">
<div class="code-block-element-YDlfvc light custom-code-block-container--plaintext hide-indicator disable-theme-style custom-code-block-container">
<div class="code-area-tZtDgG code-area" dir="ltr">
<div class="content-EGRozu code-content light-scrollbar-TbsSwi">


```plaintext
<code class="language-plaintext">net use &lt;本地盘符&gt;: \\&lt;目标IP&gt;\C$ /user:&lt;用户名&gt; &lt;密码或哈希&gt;</code>
```


</div>
</div>
</div>
</div>
<ul class="auto-hide-last-sibling-br">
 	<li><code>&lt;本地盘符&gt;</code>：本地未占用的盘符（如 Z:、Y:）；</li>
 	<li><code>&lt;目标IP&gt;</code>：远程主机的 IP 地址；</li>
 	<li><code>/user:</code>：指定登录远程主机的用户（需管理员权限）；</li>
 	<li>密码可省略，执行后会提示输入；哈希认证需结合工具（如 Impacket）使用；域管理员在域控制器上操作也不用输密码。</li>
</ul>
<h3 class="header-vfC6AV auto-hide-last-sibling-br">2. 实战示例</h3>
<div class="auto-hide-last-sibling-br paragraph-JOTKXA paragraph-element br-paragraph-space">假设目标 IP 为<code>192.168.226.8</code>，登录用户为域管理员<code>lang.com\Administrator</code>：</div>

<div class="content-EGRozu code-content light-scrollbar-TbsSwi">


```plaintext
<code class="language-plaintext"># 输入密码认证
net use Z: \\192.168.226.8\C$ /user:lang.com\Administrator

# 哈希认证（需配合工具，无需明文密码）
net use Z: \\192.168.226.8\C$ /user:lang.com\Administrator aad3b435b51404eeaad3b435b51404ee:31d6cfe0d16ae931b73c59d7e0c089c</code>
```

<h3 class="header-vfC6AV auto-hide-last-sibling-br">3. 挂载失败的常见原因</h3>
<ul class="auto-hide-last-sibling-br">
 	<li>权限不足：用户非管理员或密码错误；</li>
 	<li>防火墙拦截：远程主机 445 端口（SMB 服务）未开放；</li>
 	<li>共享关闭：目标主机通过组策略禁用了 C$ 共享；</li>
 	<li>网络问题：目标 IP 不可达或网络延迟过高。</li>
</ul>

<h2 class="header-vfC6AV auto-hide-last-sibling-br">七、安全风险与防护建议</h2>
<div class="auto-hide-last-sibling-br paragraph-JOTKXA paragraph-element br-paragraph-space">C$ 共享虽便于管理，但也可能被攻击者利用，需注意以下防护措施：</div>
<ol class="auto-hide-last-sibling-br">
 	<li><strong>限制权限</strong>：仅允许必要的管理员账号访问 C$ 共享，禁用普通用户权限；</li>
 	<li><strong>关闭不必要的共享</strong>：通过组策略禁用 C$<span class="container-X8VMJb math-inline" data-custom-copy-text="\(、admin\)">、admin$</span>等非必需共享（需谨慎，可能影响系统功能）；</li>
 	<li><strong>防火墙策略</strong>：限制 445 端口访问，仅允许信任网段连接；</li>
 	<li><strong>审计日志</strong>：开启 Windows 事件日志，监控 C$ 共享的访问和操作记录；</li>
 	<li><strong>定期检查</strong>：通过<code>net share</code>命令检查是否存在异常共享。</li>
</ol>
<h2 class="header-vfC6AV auto-hide-last-sibling-br">总结</h2>
<div class="auto-hide-last-sibling-br paragraph-JOTKXA paragraph-element br-paragraph-space">C$ 共享是 Windows 远程管理的重要功能，通过本文介绍的挂载、文件操作、程序执行等方法，可高效实现远程系统管理。但需注意，所有操作必须在合法授权的前提下进行，避免因滥用导致安全风险。掌握这些技能，无论是系统管理员还是安全从业者，都能更好地应对网络环境中的管理与防护需求。</div>
