<p class="md-end-block md-p md-focus"><span class="md-meta-i-c md-link md-expand"><a href="https://github.com/zmap/zmap"><span class="md-plain">Zmap</span></a></span><span class="md-plain"> 是一款开源的</span><span class="md-pair-s "><strong><span class="md-plain">高速网络扫描器</span></strong></span><span class="md-plain">，其设计目标是在最短的时间内对整个 IPv4 地址空间进行扫描。Zmap 采用独特的无状态（stateless）扫描方式，能够以极高的速率发送探测数据包，非常适合进行大规模网络普查和安全研究。</span></p>

<h3 class="md-end-block md-heading"><span class="md-plain">核心特点与适用场景</span></h3>
<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">速度惊人</span></strong></span><span class="md-plain">：Zmap 能够在 45 分钟内扫描完整个 IPv4 地址空间的单个端口。其高效率得益于它不维护状态连接，而是直接发送探测包并等待响应。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">专注于单一端口扫描</span></strong></span><span class="md-plain">：Zmap 的设计使其最擅长</span><span class="md-pair-s "><strong><span class="md-plain">快速确定特定端口的开放情况</span></strong></span><span class="md-plain">。它不如 Nmap 那样功能全面，但其速度是其他工具难以企及的。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">网络带宽优先</span></strong></span><span class="md-plain">：Zmap 的性能瓶颈通常在于网络带宽。为了达到最高速度，需要有足够的上行带宽支持。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">适用场景</span></strong></span><span class="md-plain">：</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">互联网资产普查</span></strong></span><span class="md-plain">：快速了解全球范围内某个端口（如 80、443、22）的开放情况。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">安全威胁研究</span></strong></span><span class="md-plain">：追踪新出现的网络漏洞或恶意软件在互联网上的传播情况。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s "><strong><span class="md-plain">学术研究</span></strong></span><span class="md-plain">：用于大规模网络测量和数据分析。</span></p>
</li>
</ul>
</li>
</ul>
<h3 class="md-end-block md-heading"><span class="md-plain">基础使用方法</span></h3>
<p class="md-end-block md-p"><span class="md-plain">Zmap 是一个命令行工具，其基本语法非常简单。</span></p>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">扫描指定端口</span></strong></span></p>



```
zmap -p 80 192.168.1.0/24
```


<p class="md-end-block md-p"><span class="md-plain">此命令会扫描 </span><span class="md-pair-s" spellcheck="false"><code>192.168.1.0/24</code></span><span class="md-plain"> 网段内所有主机的 80 端口。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">扫描整个 IPv4 地址空间</span></strong></span></p>



```
zmap -p 443 -o results.txt
```


<p class="md-end-block md-p"><span class="md-plain">这条命令会扫描全球所有主机的 443 端口，并将开放此端口的 IP 地址输出到 </span><span class="md-pair-s" spellcheck="false"><code>results.txt</code></span><span class="md-plain"> 文件中。</span></p>
</li>
</ul>

<ul class="ul-list" data-mark="-">
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">扫描整个 IPv4 地址空间的特定端口，并排除私有 IP 地址</span></strong></span></p>



```
zmap -p 80 -o results.txt --exclude-file=/etc/zmap/exclude.conf
```


<p class="md-end-block md-p"><span class="md-plain">这个命令非常重要，它能确保你不会扫描到内网地址，避免不必要的麻烦。</span></p>
</li>
 	<li class="md-list-item">
<p class="md-end-block md-p"><span class="md-pair-s"><strong><span class="md-plain">在指定带宽下，扫描特定国家的 IP 地址段</span></strong></span></p>



```
zmap -p 22 --bandwidth=10M --target-file=/root/ip_list_china.txt -o china_ssh_results.txt
```


<p class="md-end-block md-p md-focus"><span class="md-plain">通过结合 </span><span class="md-pair-s" spellcheck="false"><code>--bandwidth</code></span><span class="md-plain"> 和 </span><span class="md-pair-s" spellcheck="false"><code>--target-file</code></span><span class="md-plain md-expand"> 参数，你可以精确控制扫描的速率和范围，实现对特定区域的精准扫描。</span></p>
</li>
</ul>
